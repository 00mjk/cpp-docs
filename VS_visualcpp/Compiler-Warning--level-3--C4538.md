# Compiler Warning (level 3) C4538
'type' : const/volatile qualifiers on this type are not supported  
  
 A qualifier keyword was applied to an array incorrectly. For more information, see [array](../VS_visualcpp/Arrays--C---Component-Extensions-.md).  
  
 The following sample generates C4538:  
  
```  
// C4538.cpp  
// compile with: /clr /W3 /LD  
const array<int> ^f1();   // C4538  
array<const int> ^f2();   // OK  
```
