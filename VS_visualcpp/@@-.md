# @@:
Defines a code label recognizable only between *label1* and *label2*, where *label1* is either start of code or the previous `@@:` label, and *label2* is either end of code or the next `@@:` label. See [@B](../VS_visualcpp/@B.md) and [@F](../VS_visualcpp/@F.md).  
  
## Syntax  
  
```  
  
@@:  
  
```
