# MMC Snap-In Classes
The following classes provide support for developing Microsoft Management Console (MMC) snap-in components:  
  
-   [CSnapInItemImpl](../VS_visualcpp/CSnapInItemImpl-Class.md) Implements a snap-in node object, such as adding menu items and toolbars, and forwarding commands for the snap-in node to the appropriate handler function.  
  
-   [CSnapInPropertyPageImpl](../VS_visualcpp/CSnapInPropertyPageImpl-Class.md) Implements a snap-in property page object.
