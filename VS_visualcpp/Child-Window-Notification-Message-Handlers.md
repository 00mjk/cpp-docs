# Child Window Notification Message Handlers
There are five categories of child window notification messages:  
  
|Category|Description|  
|--------------|-----------------|  
|[Generic Control Handler](../VS_visualcpp/Generic-Control-Handler.md)|Handler for generic control notification codes.|  
|[User Button Handlers](../VS_visualcpp/User-Button-Handlers.md)|Handlers for user button notification codes.|  
|[Combo Box Handlers](../VS_visualcpp/Combo-Box-Handlers.md)|Handlers for combo box notification codes.|  
|[Edit Control Handlers](../VS_visualcpp/Edit-Control-Handlers.md)|Handlers for edit control notification codes.|  
|[List Box Handlers](../VS_visualcpp/List-Box-Handlers.md)|Handlers for list box notification codes.|
