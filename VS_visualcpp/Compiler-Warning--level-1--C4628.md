# Compiler Warning (level 1) C4628
digraphs not supported with -Ze. Character sequence 'digraph' not interpreted as alternate token for 'char'  
  
 Digraphs are not supported under [/Ze](../VS_visualcpp/-Za---Ze--Disable-Language-Extensions-.md). This warning will be followed by an error.  
  
 This warning is off by default. See [Compiler Warnings That Are Off by Default](../VS_visualcpp/Compiler-Warnings-That-Are-Off-by-Default.md) for more information.  
  
 The following sample generates C4628:  
  
```  
// C4628.cpp  
// compile with: /WX  
#pragma warning(default : 4628)  
int main()  
<%   // C4628 <% digraph for {  
}  
```
