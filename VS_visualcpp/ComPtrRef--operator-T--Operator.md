# ComPtrRef::operator T* Operator
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
operator T*();  
```  
  
## Remarks  
 Returns the value of the [ptr_](../VS_visualcpp/ComPtrRefBase--ptr_-Data-Member.md) data member of the current ComPtrRef object.  
  
## Requirements  
 **Header:** client.h  
  
 **Namespace:** Microsoft::WRL::Details
