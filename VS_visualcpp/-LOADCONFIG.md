# -LOADCONFIG
```  
/LOADCONFIG  
```  
  
## Remarks  
 This option dumps the IMAGE_LOAD_CONFIG_DIRECTORY structure, an optional structure that is used by the Windows NT loader and defined in WINNT.H.  
  
 Only the [/HEADERS](../VS_visualcpp/-HEADERS.md) DUMPBIN option is available for use on files produced with the [/GL](../VS_visualcpp/-GL--Whole-Program-Optimization-.md) compiler option.
