# ML Nonfatal Error A2004
**symbol type conflict : identifier**  
  
 The [EXTERNDEF](../VS_visualcpp/EXTERNDEF.md) or [LABEL](../VS_visualcpp/LABEL--MASM-.md) directive was used on a variable, symbol, data structure, or label that was defined in the same module but with a different type.
