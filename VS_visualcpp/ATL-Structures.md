# ATL Structures
The Active Template Library includes the following structures.  
  
## In This Section  
 [ATL_DRAWINFO](../VS_visualcpp/ATL_DRAWINFO-Structure.md)  
 Contains information used for rendering to various targets, such as a printer, metafile, or ActiveX control.  
  
 [_AtlCreateWndData](../VS_visualcpp/_AtlCreateWndData-Structure.md)  
 Contains class instance data in windowing code in ATL.  
  
 [_ATL_BASE_MODULE70](../VS_visualcpp/_ATL_BASE_MODULE70-Structure.md)  
 Used by any project that uses ATL.  
  
 [_ATL_COM_MODULE70](../VS_visualcpp/_ATL_COM_MODULE70-Structure.md)  
 Used by COM-related code in ATL.  
  
 [_ATL_FUNC_INFO](../VS_visualcpp/_ATL_FUNC_INFO-Structure.md)  
 Contains type information used to describe a method or property on a dispinterface.  
  
 [_ATL_MODULE70](../VS_visualcpp/_ATL_MODULE70-Structure.md)  
 Contains data used by every ATL module.  
  
 [_ATL_WIN_MODULE70](../VS_visualcpp/_ATL_WIN_MODULE70-Structure.md)  
 Used by windowing code in ATL.
