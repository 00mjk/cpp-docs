# COM Modules Classes
The following classes provide support for a COM module:  
  
-   [CAtlBaseModule](../VS_visualcpp/CAtlBaseModule-Class.md) This class is instantiated in every ATL project.  
  
-   [CAtlComModule](../VS_visualcpp/CAtlComModule-Class.md) This class implements a COM server module.  
  
-   [CAtlModule](../VS_visualcpp/CAtlModule-Class.md) This class provides methods used by several ATL module classes.  
  
-   [CAtlModuleT](../VS_visualcpp/CAtlModuleT-Class.md) This class implements an ATL module.  
  
-   [CAtlExeModuleT](../VS_visualcpp/CAtlExeModuleT-Class.md) This class represents the module for an application.  
  
-   [CAtlServiceModuleT](../VS_visualcpp/CAtlServiceModuleT-Class.md) This class implements a service.  
  
-   [CAtlWinModule](../VS_visualcpp/CAtlWinModule-Class.md) This class provides support for ATL windowing components.  
  
-   [CComModule](../VS_visualcpp/CComModule-Class.md) This class implements a DLL or EXE module. Obsolete in ATL 7.0.  
  
-   [CComAutoThreadModule](../VS_visualcpp/CComAutoThreadModule-Class.md) This class implements an EXE module, with support for multiple thread-pooled apartments. Obsolete in ATL 7.0.
