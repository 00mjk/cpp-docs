# rename_search_namespace
**C++ Specific**  
  
 Has the same functionality as the [rename_namespace](../VS_visualcpp/rename_namespace.md) attribute but is used on type libraries that you use the #import directive with the [auto_search](../VS_visualcpp/auto_search.md) attribute.  
  
## Syntax  
  
```  
rename_search_namespace("NewName")  
```  
  
#### Parameters  
 `NewName`  
 The new name of the namespace.
