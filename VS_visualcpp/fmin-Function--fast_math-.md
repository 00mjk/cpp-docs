# fmin Function (fast_math)
Determine the minimum numeric value of the arguments  
  
## Syntax  
  
```  
inline float min(  
    float _X,  
    float _Y ) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Integer value  
  
 `_Y`  
 Integer value  
  
## Return Value  
 Return the minimum numeric value of the arguments  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
