# .LISTALL
Starts listing of all statements.  
  
## Syntax  
  
```  
  
.LISTALL  
  
```  
  
## Remarks  
 Equivalent to the combination of [.LIST](../VS_visualcpp/.LIST.md), [.LISTIF](../VS_visualcpp/.LISTIF.md), and [.LISTMACROALL](../VS_visualcpp/.LISTMACROALL.md).
