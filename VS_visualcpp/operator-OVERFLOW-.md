# operator OVERFLOW?
Status of overflow flag. Used only within [.IF](../VS_visualcpp/.IF.md), [.WHILE](../VS_visualcpp/.WHILE.md), or [.REPEAT](../VS_visualcpp/.REPEAT.md) blocks and evaluated at run time, not at assembly time.  
  
## Syntax  
  
```  
  
OVERFLOW?  
  
```
