# .FARDATA?
When used with [.MODEL](../VS_visualcpp/.MODEL.md), starts a far data segment for uninitialized data (segment name FAR_BSS or *name*).  
  
## Syntax  
  
```  
  
.FARDATA? [[name]]  
```
