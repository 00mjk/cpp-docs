# Class Factories Classes
The following classes implement or support a class factory:  
  
-   [CComClassFactory](../VS_visualcpp/CComClassFactory-Class.md) Provides a default class factory for object creation.  
  
-   [CComClassFactory2](../VS_visualcpp/CComClassFactory2-Class.md) Controls object creation through a license.  
  
-   [CComClassFactoryAutoThread](../VS_visualcpp/CComClassFactoryAutoThread-Class.md) Allows objects to be created in multiple thread-pooled apartments.  
  
-   [CComClassFactorySingleton](../VS_visualcpp/CComClassFactorySingleton-Class.md) Creates a single object.  
  
-   [CComCoClass](../VS_visualcpp/CComCoClass-Class.md) Defines the class factory for the object.
