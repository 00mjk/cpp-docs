# AFX_GLOBAL_DATA::bIsOSAlphaBlendingSupport
Indicates whether the operating system supports alpha blending.  
  
## Syntax  
  
```  
BOOL  bIsOSAlphaBlendingSupport;  
```  
  
## Remarks  
 `TRUE` indicates alpha blending is supported; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxglobals.h
