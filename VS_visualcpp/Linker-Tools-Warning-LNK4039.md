# Linker Tools Warning LNK4039
section 'name' specified with /SECTION option does not exist  
  
 [DUMPBIN](../VS_visualcpp/DUMPBIN-Reference.md) or [EDITBIN](../VS_visualcpp/EDITBIN-Reference.md) could not find a section called `name` in the input file.  
  
 To see the sections in a file, run DUMPBIN with the [/HEADERS](../VS_visualcpp/-HEADERS.md) option.
