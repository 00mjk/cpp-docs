# sqrtf Function (fast_math)
Calculates the squre root of the argument  
  
## Syntax  
  
```  
inline float sqrtf(    float _X ) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the squre root of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
