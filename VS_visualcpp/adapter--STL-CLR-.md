# adapter (STL-CLR)
The STL/CLR header `<cliext/adapter>` specifies two template classes (`collection_adapter` and `range_adapter`), and the template function `make_collection`.  
  
## Syntax  
  
```  
#include <cliext/adapter>  
```  
  
## Remarks  
  
|Class|Description|  
|-----------|-----------------|  
|[collection_adapter (STL/CLR)](../VS_visualcpp/collection_adapter--STL-CLR-.md)|Wraps the Base Class Library (BCL) collection as a range.|  
|[range_adapter (STL/CLR)](../VS_visualcpp/range_adapter--STL-CLR-.md)|Wraps the range as a BCL collection.|  
  
|Function|Description|  
|--------------|-----------------|  
|[make_collection (STL/CLR)](../VS_visualcpp/make_collection--STL-CLR-.md)|Creates a range adapter using an iterator pair.|  
  
## Requirements  
 **Header:** <cliext/adapter>  
  
 **Namespace:** cliext
