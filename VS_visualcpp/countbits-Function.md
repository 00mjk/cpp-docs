# countbits Function
Counts the number of set bits in _X  
  
## Syntax  
  
```  
inline unsigned int countbits(    unsigned int _X ) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Unsigned integer value  
  
## Return Value  
 Returns the number of set bits in _X  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency::direct3d
