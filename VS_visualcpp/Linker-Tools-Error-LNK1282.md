# Linker Tools Error LNK1282
unable to /REBASE file; it has been signed  
  
 You attempted to change the base address of a signed assembly with the /REBASE option for [editbin](../VS_visualcpp/EDITBIN-Reference.md). To do this, first change the base address and then sign the assembly.
