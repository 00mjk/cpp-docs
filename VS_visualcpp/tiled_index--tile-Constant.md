# tiled_index::tile Constant
Stores an [index](../VS_visualcpp/index-Class.md) object of rank 1, 2, or 3 that represents the coordinates of the current tile of a [tiled_extent](../VS_visualcpp/tiled_extent-Class.md) object.  
  
## Syntax  
  
```  
const index<rank> tile;  
```  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency
