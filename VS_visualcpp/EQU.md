# EQU
The first directive assigns numeric value of *expression* to *name*.  
  
## Syntax  
  
```  
  
      name EQU expression  
name EQU <text>  
```  
  
## Remarks  
 The *name* cannot be redefined later.  
  
 The second directive assigns specified *text* to *name*. The *name* can be assigned a different *text* later. See [TEXTEQU](../VS_visualcpp/TEXTEQU.md).
