# Compiler Error C3276
'keyword' : jump out of __finally/finally block has undefined behavior during termination handling  
  
 This error is the same as the [C4532](../VS_visualcpp/Compiler-Warning--level-1--C4532.md) warning. However, when you are using /clr, this condition cannot be disabled with the [warning](../VS_visualcpp/warning.md) pragma.
