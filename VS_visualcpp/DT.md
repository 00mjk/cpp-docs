# DT
Allocates and optionally initializes 10 bytes of storage for each `initializer`. Can also be used as a type specifier anywhere a type is legal. `DT` is a synonym of [TBYTE](../VS_visualcpp/TBYTE.md).  
  
## Syntax  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
