# round Function
Rounds _X to the nearest integer  
  
## Syntax  
  
```  
inline float round(  
    float _X ) restrict(amp);  
  
inline double round(  
    double _X ) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the nearest integer of _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
