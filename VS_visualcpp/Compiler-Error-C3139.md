# Compiler Error C3139
'struct' : cannot export a UDT without members  
  
 You attempted to apply the [export](../VS_visualcpp/export.md) attribute to an empty UDT (user-defined type). For example:  
  
```  
// C3139.cpp  
#include "unknwn.h"  
[emitidl];  
[module(name=xx)];  
  
[export] struct MyStruct {   // C3139 empty type  
};  
int main(){}  
```
