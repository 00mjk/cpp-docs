To find an ATL macro by category, see the following topics.  
  
 [Aggregation and Class Factory Macros](../VS_visualcpp/Aggregation-and-Class-Factory-Macros.md)

 [String Conversion Macros](../Topic/ATL%20and%20MFC%20String%20Conversion%20Macros.md)

 [Window Class Macros](../VS_visualcpp/Window-Class-Macros.md)

 [Windows Messages Macros](../VS_visualcpp/Windows-Messages-Macros.md)
