# &lt;para&gt; (Visual C++)
The <para\> tag is for use inside a tag, such as [<summary\>](../VS_visualcpp/-summary---Visual-C---.md), [<remarks\>](../VS_visualcpp/-remarks---Visual-C---.md), or [<returns\>](../VS_visualcpp/-returns---Visual-C---.md), and lets you add structure to the text.  
  
## Syntax  
  
```  
<para>content</para>  
```  
  
#### Parameters  
 `content`  
 The text of the paragraph.  
  
## Remarks  
 Compile with [/doc](../VS_visualcpp/-doc--Process-Documentation-Comments---C-C---.md) to process documentation comments to a file.  
  
## Example  
 See [<summary\>](../VS_visualcpp/-summary---Visual-C---.md) for an example of using <para\>.
