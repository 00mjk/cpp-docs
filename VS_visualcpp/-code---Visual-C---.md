# &lt;code&gt; (Visual C++)
The <code\> tag gives you a way to indicate one or more lines as code.  
  
## Syntax  
  
```  
<code>content</code>  
```  
  
#### Parameters  
 `content`  
 The text you want marked as code.  
  
## Remarks  
 Use [<c\>](../VS_visualcpp/-c---Visual-C---.md) to indicate a portion of text should be marked as code.  
  
 Compile with [/doc](../VS_visualcpp/-doc--Process-Documentation-Comments---C-C---.md) to process documentation comments to a file.
