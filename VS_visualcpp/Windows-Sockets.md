# Windows Sockets
This family of articles covers the MFC implementation of Windows Sockets. MFC supplies two classes to support programming network applications with the Windows Sockets API. Class [CAsyncSocket](../VS_visualcpp/CAsyncSocket-Class.md) encapsulates the Windows Sockets API one for one, giving advanced network programmers the most power and flexibility. Class [CSocket](../VS_visualcpp/CSocket-Class.md) provides a simplified interface for serializing data to and from a [CArchive](../VS_visualcpp/CArchive-Class.md) object.
