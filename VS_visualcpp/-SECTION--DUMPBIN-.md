# -SECTION (DUMPBIN)
```  
/SECTION:section  
```  
  
## Remarks  
 This option restricts the output to information on the specified *section*. Use the [/HEADERS](../VS_visualcpp/-HEADERS.md) option to get a list of sections in the file.  
  
 Only the [/HEADERS](../VS_visualcpp/-HEADERS.md) DUMPBIN option is available for use on files produced with the [/GL](../VS_visualcpp/-GL--Whole-Program-Optimization-.md) compiler option.
