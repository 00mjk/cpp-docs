# IF2
**IF** block is evaluated on every assembly pass if **OPTION:SETIF2** is **TRUE**.  
  
## Syntax  
  
```  
  
IF2   
expression  
  
```  
  
## Remarks  
 See [IF](../VS_visualcpp/IF--MASM-.md) for complete syntax.
