# no_search_namespace
**C++ Specific**  
  
 Has the same functionality as the [no_namespace](../VS_visualcpp/no_namespace.md) attribute but is used on type libraries that you use the #import directive with the [auto_search](../VS_visualcpp/auto_search.md) attribute.  
  
## Syntax  
  
```  
no_search_namespace  
```
