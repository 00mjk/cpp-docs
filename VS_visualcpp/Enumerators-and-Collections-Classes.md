# Enumerators and Collections Classes
The following classes provide support for COM collections and enumerations:  
  
-   [CComEnum](../VS_visualcpp/CComEnum-Class.md) Defines a COM enumerator object based on an array.  
  
-   [CComEnumImpl](../VS_visualcpp/CComEnumImpl-Class.md) Provides the implementation for a COM enumerator interface where the items being enumerated are stored in an array.  
  
-   [CComEnumOnSTL](../VS_visualcpp/CComEnumOnSTL-Class.md) Defines a COM enumerator object based on an STL collection.  
  
-   [IEnumOnSTLImpl](../VS_visualcpp/IEnumOnSTLImpl-Class.md) Provides the implementation for a COM enumerator interface where the items being enumerated are stored in an STL-compatible container.  
  
-   [ICollectionOnSTLImpl](../VS_visualcpp/ICollectionOnSTLImpl-Class.md) Provides the implementation for the **Count**, **Item**, and `_NewEnum` properties of a collection interface.
