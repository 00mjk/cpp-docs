# Compiler COM Global Functions
**Microsoft Specific**  
  
 The following routines are available:  
  
|Function|Description|  
|--------------|-----------------|  
|[_com_raise_error](../VS_visualcpp/_com_raise_error.md)|Throws a [_com_error](../VS_visualcpp/_com_error-Class.md) in response to a failure.|  
|[_set_com_error_handler](../VS_visualcpp/_set_com_error_handler.md)|Replaces the default function that is used for COM error-handling.|  
|[ConvertBSTRToString](../VS_visualcpp/ConvertBSTRToString.md)|Converts a `BSTR` value to a **char \***.|  
|[ConvertStringToBSTR](../VS_visualcpp/ConvertStringToBSTR.md)|Converts a **char \*** value to a `BSTR`.|  
  
## END Microsoft Specific
