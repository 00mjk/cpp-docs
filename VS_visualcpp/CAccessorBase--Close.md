# CAccessorBase::Close
Closes the accessors.  
  
## Syntax  
  
```  
  
void Close( );  
  
```  
  
## Remarks  
 You must call [ReleaseAccessors](../VS_visualcpp/CAccessorBase--ReleaseAccessors.md) first.
