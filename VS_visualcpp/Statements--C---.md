# Statements (C++)
C++ statements are the program elements that control how and in what order objects are manipulated. This section includes:  
  
-   [Overview](../VS_visualcpp/Overview-of-C---Statements.md)  
  
-   [Labeled Statements](../VS_visualcpp/Labeled-Statements.md)  
  
-   Categories of Statements  
  
    -   [Expression statements](../VS_visualcpp/Expression-Statement.md). These statements evaluate an expression for its side effects or for its return value.  
  
    -   [Null statements](../VS_visualcpp/Null-Statement.md). These statements can be provided where a statement is required by the C++ syntax but where no action is to be taken.  
  
    -   [Compound statements](../VS_visualcpp/Compound-Statements--Blocks-.md). These statements are groups of statements enclosed in curly braces ({ }). They can be used wherever a single statement may be used.  
  
    -   [Selection statements](../VS_visualcpp/Selection-Statements--C---.md). These statements perform a test; they then execute one section of code if the test evaluates to true (nonzero). They may execute another section of code if the test evaluates to false.  
  
    -   [Iteration statements](../VS_visualcpp/Iteration-Statements--C---.md). These statements provide for repeated execution of a block of code until a specified termination criterion is met.  
  
    -   [Jump statements](../VS_visualcpp/Jump-Statements--C---.md). These statements either transfer control immediately to another location in the function or return control from the function.  
  
    -   [Declaration statements](assetId:///14538558-356f-450e-9e1e-3cd62ba952b9). Declarations introduce a name into a program. ([Declarations](../Topic/Declarations.md) provides more detailed information about declarations.)  
  
 For information on exception handling statements see [Exception Handling](../VS_visualcpp/Exception-Handling-in-Visual-C--.md).
