# Implementing a Window
Class [CWindowImpl](../VS_visualcpp/CWindowImpl-Class.md) allows you to implement a window and handle its messages. Message handing in ATL is based on a message map. This section explains:  
  
-   How to [add a message handler](../VS_visualcpp/Adding-an-ATL-Message-Handler.md) to a control.  
  
-   What [message maps](../VS_visualcpp/Message-Maps--ATL-.md) are and how to use them.  
  
-   The syntax for [message handler functions](../VS_visualcpp/Message-Handler-Functions.md).  
  
-   How to [implement a window with CWindowImpl](../VS_visualcpp/Implementing-a-Window-with-CWindowImpl.md).
