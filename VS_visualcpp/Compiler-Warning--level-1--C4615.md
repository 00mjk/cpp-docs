# Compiler Warning (level 1) C4615
\#pragma warning : unknown user warning type  
  
 An invalid warning specifier was used with **pragma** [warning](../VS_visualcpp/warning.md). To resolve the error, use a valid warning specifier.  
  
 The following sample generates C4615:  
  
```  
// C4615.cpp  
// compile with: /W1 /LD  
#pragma warning(enable : 4401)   // C4615, 'enable' not valid specifier  
  
// use the code below to resolve the error  
// #pragma warning(default : 4401)  
```
