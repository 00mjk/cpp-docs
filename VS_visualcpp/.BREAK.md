# .BREAK
Generates code to terminate a [.WHILE](../VS_visualcpp/.WHILE.md) or [.REPEAT](../VS_visualcpp/.REPEAT.md) block if *condition* is true.  
  
## Syntax  
  
```  
  
.BREAK [[.IF condition]]   
```
