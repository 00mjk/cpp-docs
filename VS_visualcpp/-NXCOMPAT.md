# -NXCOMPAT
```  
/NXCOMPAT[:NO]  
```  
  
## Remarks  
 Indicates that an executable was tested to be compatible with the Windows Data Execution Prevention feature.  
  
 For more information, see [/NXCOMPAT (Compatible with Data Execution Prevention)](../VS_visualcpp/-NXCOMPAT--Compatible-with-Data-Execution-Prevention-.md).
