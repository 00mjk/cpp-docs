# DDX_DHtml Helper Macros
The DDX_DHtml helper macros allow easy access to the commonly used properties of controls on an HTML page.  
  
### Data Exchange Macros  
  
|||  
|-|-|  
|[DDX_DHtml_ElementValue](../VS_visualcpp/DDX_DHtml_ElementValue.md)|Sets or retrieves the Value property from the selected control.|  
|[DDX_DHtml_ElementInnerText](../VS_visualcpp/DDX_DHtml_ElementInnerText.md)|Sets or retrieves the text between the start and end tags of the current element.|  
|[DDX_DHtml_ElementInnerHtml](../VS_visualcpp/DDX_DHtml_ElementInnerHtml.md)|Sets or retrieves the HTML between the start and end tags of the current element.|  
|[DDX_DHtml_Anchor_Href](../VS_visualcpp/DDX_DHtml_Anchor_Href.md)|Sets or retrieves the destination URL or anchor point.|  
|[DDX_DHtml_Anchor_Target](../VS_visualcpp/DDX_DHtml_Anchor_Target.md)|Sets or retrieves the target window or frame.|  
|[DDX_DHtml_Img_Src](../VS_visualcpp/DDX_DHtml_Img_Src.md)|Sets or retrieves the name of an image or a video clip in the document.|  
|[DDX_DHtml_Frame_Src](../VS_visualcpp/DDX_DHtml_Frame_Src.md)|Sets or retrieves the URL of the associated frame.|  
|[DDX_DHtml_IFrame_Src](../VS_visualcpp/DDX_DHtml_IFrame_Src.md)|Sets or retrieves the URL of the associated frame.|  
  
## Requirements  
 **Header:** afxdhtml.h
