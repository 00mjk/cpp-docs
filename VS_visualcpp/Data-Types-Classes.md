# Data Types Classes
The following classes wrap C++ data types:  
  
-   [CComBSTR](../VS_visualcpp/CComBSTR-Class.md) Wraps the `BSTR` data type.  
  
-   [CComVariant](../VS_visualcpp/CComVariant-Class.md) Wraps the **VARIANT** data type.  
  
-   [CComCurrency](../VS_visualcpp/CComCurrency-Class.md) Includes methods and operators for creating and managing a `CURRENCY` object.
