# C++ Standard Library Overview
All C++ library entities are declared or defined in one or more standard headers. This implementation includes two additional headers, `<hash_map>` and `<hash_set>`, that are not required by the C++ Standard. For a complete list of headers that this implementation supports, see [Header Files Reference](../VS_visualcpp/C---Standard-Library-Header-Files.md).  
  
 A freestanding implementation of the C++ library provides only a subset of these headers:  
  
|||  
|-|-|  
|[<cstddef\>](../VS_visualcpp/-cstddef-.md)|[<cstdlib\>](../VS_visualcpp/-cstdlib-.md) (declaring at least the functions `abort`, `atexit`, and `exit`)|  
|[<exception\>](../VS_visualcpp/-exception-.md)|[<limits\>](../VS_visualcpp/-limits-.md)|  
|[<new\>](../VS_visualcpp/-new-.md)|[<cstdarg\>](../VS_visualcpp/-cstdarg-.md)|  
  
 The C++ library headers have two broader subdivisions:  
  
-   [iostreams](../VS_visualcpp/iostreams-Conventions.md) conventions.  
  
-   [Standard Template Library](../Topic/Standard%20Template%20Library.md) conventions.
