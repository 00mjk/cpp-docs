# .NOCREF
Suppresses listing of symbols in the symbol table and browser file.  
  
## Syntax  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## Remarks  
 If names are specified, then only the given names are suppressed. Same as [.XCREF](../VS_visualcpp/.XCREF.md).
