# Supporting Navigation in a Record View  (MFC Data Access)
This topic explains how to support movement from record to record in your record view, including information about:  
  
-   [Command handling](../VS_visualcpp/Command-Handlers-for-Record-Scrolling---MFC-Data-Access-.md) for record scrolling commands.  
  
-   [User-interface update handlers for scrolling](../VS_visualcpp/User-Interface-Updating-for-Record-Views---MFC-Data-Access-.md) commands.  
  
 The information in these topics applies to both [CRecordView](../VS_visualcpp/CRecordView-Class.md) (ODBC) and [CDaoRecordView](../VS_visualcpp/CDaoRecordView-Class.md) (DAO).
