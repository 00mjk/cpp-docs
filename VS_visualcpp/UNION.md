# UNION
Declares a union of one or more data types. The *fielddeclarations* must be valid data definitions. Omit the [ENDS](../VS_visualcpp/ENDS--MASM-.md) *name* label on nested **UNION** definitions.  
  
## Syntax  
  
```  
  
      name   
      UNION [[alignment]] [[, NONUNIQUE]]  
   fielddeclarations  
[[name]] ENDS  
```
