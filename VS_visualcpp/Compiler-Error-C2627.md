# Compiler Error C2627
'function' : member function not allowed in anonymous union  
  
 An [anonymous union](../Topic/Anonymous%20Unions.md) cannot have member functions.  
  
 The following sample generates C2627:  
  
```  
// C2627.cpp  
int main() {  
   union { void f(){} };   // C2627  
   union X { void f(){} };  
}  
```
