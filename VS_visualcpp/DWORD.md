# DWORD
Allocates and optionally initializes a double word (4 bytes) of storage for each `initializer`.  
  
## Syntax  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## Remarks  
 Can also be used as a type specifier anywhere a type is legal.
