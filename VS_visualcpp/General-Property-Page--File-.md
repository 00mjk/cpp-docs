# General Property Page (File)
When a file is selected in **Solution Explorer**, the **General** property page under the **Configuration Properties** node contains the following properties:  
  
 **Exclude From Build**  
 Specifies whether the file should be in the build for the current configuration.  
  
 To programmatically access this property, see <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.ExcludedFromBuild?qualifyHint=False>.  
  
 **Tool**  
 The tool that will be used to build this file. See [Specifying Custom Build Tools](../VS_visualcpp/Specifying-Custom-Build-Tools.md) for more information.  
  
 To programmatically access this property, see <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.Tool?qualifyHint=False>.  
  
 For information on how to access the **General** property page under the **Configuration Properties** node, see [How To: Specify Project Properties with Property Pages](../Topic/How%20to:%20Specify%20Project%20Properties%20with%20Property%20Pages.md).
