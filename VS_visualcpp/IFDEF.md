# IFDEF
Grants assembly if *name* is a previously defined label, variable, or symbol.  
  
## Syntax  
  
```  
  
IFDEF name  
```  
  
## Remarks  
 See [IF](../VS_visualcpp/IF--MASM-.md) for complete syntax.
