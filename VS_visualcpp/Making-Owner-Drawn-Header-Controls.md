# Making Owner-Drawn Header Controls
You can define individual items of a header control ([CHeaderCtrl](../VS_visualcpp/CHeaderCtrl-Class.md)) to be owner-drawn items. For more information, see [Owner-Drawn Header Controls](http://msdn.microsoft.com/library/windows/desktop/bb775238) in the Windows SDK.
