# try_to_lock_t Structure
Represents a `struct` that defines a [try_to_lock](../VS_visualcpp/-mutex--functions.md#try_to_lock_variable) object. Used to select one of the overloaded constructors of [unique_lock](../VS_visualcpp/unique_lock-Class.md).  
  
## Syntax  
  
```cpp  
struct try_to_lock_t;  
```  
  
## Requirements  
 **Header:** mutex  
  
 **Namespace:** std
