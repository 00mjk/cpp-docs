# Date and Time: Database Support
As of version 4.0, MFC database programming uses the [COleDateTime](../VS_visualcpp/COleDateTime-Class.md) and [COleDateTimeSpan](../VS_visualcpp/COleDateTimeSpan-Class.md) classes to represent date and time data. These classes, also used in Automation, are derived from class [COleVariant](../VS_visualcpp/COleVariant-Class.md). They supply better support for managing date and time data than do [CTime](../VS_visualcpp/CTime-Class.md) and [CTimeSpan](../VS_visualcpp/CTimeSpan-Class.md).
