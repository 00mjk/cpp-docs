# Attributes by Usage
This topic lists attributes according to the C++ language elements to which they apply.  
  
 If an attribute precedes a Visual C++ element that is not in the attribute's scope, the attribute block is treated as a comment.  
  
|Attribute|Description|  
|---------------|-----------------|  
|[Module Attributes](../VS_visualcpp/Module-Attributes.md)|Applies to the [module](../VS_visualcpp/module--C---.md) attribute.|  
|[Interface Attributes](../VS_visualcpp/Interface-Attributes.md)|Applies to the [__interface](../VS_visualcpp/__interface.md) C++ keyword.|  
|[Class Attributes](../VS_visualcpp/Class-Attributes.md)|Applies to the C++ keyword.|  
|[Method Attributes](../VS_visualcpp/Method-Attributes.md)|Applies to the methods in a class, coclass, or interface.|  
|[Parameter Attributes](../VS_visualcpp/Parameter-Attributes.md)|Applies to parameters of a method in a class or interface.|  
|[Data Member Attributes](../VS_visualcpp/Data-Member-Attributes.md)|Applies to the data members in a class, coclass, or interface.|  
|[Typedef, Enum, Union, and Struct Attributes](../VS_visualcpp/Typedef--Enum--Union--and-Struct-Attributes.md)|Applies to the C++ keywords.|  
|[Array Attributes](../VS_visualcpp/Array-Attributes.md)|Applies to arrays or **SAFEARRAY**s.|  
|[Stand-Alone Attributes](../VS_visualcpp/Stand-Alone-Attributes.md)|Operates more like a line of code but does not operate on a C++ keyword. Stand-alone attribute statements require a semicolon at the end of the line.|  
|[Custom Attributes](../VS_visualcpp/Custom-Attributes--C---.md)|Allows the user to extend metadata.|
