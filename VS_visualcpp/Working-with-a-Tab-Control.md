# Working with a Tab Control
The easiest way to use a tab control ([CTabCtrl](../VS_visualcpp/CTabCtrl-Class.md)) is by adding it to a dialog template resource with the dialog editor. You can also use a tab control by itself. MFC calls **InitCommonControls** for you. The key tasks are as follows:  
  
-   [Creating the tab control](../VS_visualcpp/Creating-the-Tab-Control.md)  
  
-   [Adding tabs to a tab control](../VS_visualcpp/Adding-Tabs-to-a-Tab-Control.md)  
  
-   [Processing tab control notification messages](../VS_visualcpp/Processing-Tab-Control-Notification-Messages.md)  
  
 If the tab control object is embedded in a parent view or dialog class, the control is destroyed when the parent is destroyed.
