# Fatal Error C1077
compiler limit : cannot have more than number command line options  
  
 The number of command-line options exceeds the internal limit.  
  
 There may be too many symbols defined with [/D](../VS_visualcpp/-D--Preprocessor-Definitions-.md). (Place the definitions in a header file instead.)
