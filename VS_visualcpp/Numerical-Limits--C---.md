# Numerical Limits (C++)
The two standard include files, LIMITS.H and FLOAT.H, define the numerical limits, or minimum and maximum values that a variable of a given type can hold. These minimums and maximums are guaranteed to be portable to any C++ compiler that uses the same data representation as ANSI C. The LIMITS.H include file defines the [numerical limits for integral types](../VS_visualcpp/Integer-Limits.md), and FLOAT.H defines the [numerical limits for floating types](../VS_visualcpp/Floating-Limits.md).
