# operator IMAGEREL
Returns the image relative offset of `expression`.  
  
## Syntax  
  
```  
IMAGEREL expression  
```  
  
## Remarks  
 The resulting value is often referred to as an RVA or Relative Virtual Address.  
  
 IMAGEREL is available only with COFF object emission.
