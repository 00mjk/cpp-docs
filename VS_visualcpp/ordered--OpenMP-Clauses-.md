# ordered (OpenMP Clauses)
Required on a parallel [for](../VS_visualcpp/for--OpenMP-.md) statement if an [ordered](../VS_visualcpp/ordered--OpenMP-Directives-.md) directive is to be used in the loop.  
  
## Syntax  
  
```  
ordered  
```  
  
## Remarks  
 `ordered` applies to the [for](../VS_visualcpp/for--OpenMP-.md) directive.  
  
 For more information, see [2.4.1 for Construct](../VS_visualcpp/2.4.1-for-Construct.md).  
  
## Example  
 See [ordered](../VS_visualcpp/ordered--OpenMP-Directives-.md) for an example of using `ordered` clause.
