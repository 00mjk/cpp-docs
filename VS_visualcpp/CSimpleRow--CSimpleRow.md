# CSimpleRow::CSimpleRow
The constructor.  
  
## Syntax  
  
```  
  
      CSimpleRow(  
   DBCOUNTITEM iRowsetCur   
);  
```  
  
#### Parameters  
 `iRowsetCur`  
 [in] Index to the current rowset.  
  
## Remarks  
 Sets [m_iRowset](../VS_visualcpp/CSimpleRow--m_iRowset.md) to `iRowsetCur`.

 **Header:** atldb.h
