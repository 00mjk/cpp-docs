# Compiler Warning (level 4) C4690
[ emitidl( pop ) ] : more pops than pushes  
  
 The [emitidl](../VS_visualcpp/emitidl.md) attribute was popped one more time that it was pushed.  
  
## Example  
 The following sample generates C4690.  
  
```  
// C4690.cpp  
// compile with: /c /W4  
[emitidl(pop)];   // C4690  
class x {};  
```
