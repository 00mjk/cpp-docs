# OLE DB Classes
The OLE DB support in MFC currently consists of the class [COLEDBRecordView](../VS_visualcpp/COleDBRecordView-Class.md). `COleDBRecordView` displays database records in controls, through a form view directly connected to a [CRowset](../VS_visualcpp/CRowset-Class.md) object. For more information about the OLE DB consumer templates, see [List of OLE DB Consumer Templates](../VS_visualcpp/OLE-DB-Consumer-Templates-Reference.md).
