# Compiler Warning (level 1) C4329
__declspec(align()) is ignored on enum  
  
 Use of the [align](../VS_visualcpp/align--C---.md) keyword of the [__declspec](../VS_visualcpp/__declspec.md) modifier is not allowed on an `enum`. The following sample generates C4329:  
  
```  
// C4329.cpp  
// compile with: /W1 /LD  
enum __declspec(align(256)) TestEnum {   // C4329  
   TESTVAL1,  
   TESTVAL2,  
   TESTVAL3  
};  
__declspec(align(256)) enum TestEnum1;  
```
