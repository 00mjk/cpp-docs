# ML Nonfatal Error A2019
**operand must be RECORD type or field**  
  
 The operand following the [WIDTH](../VS_visualcpp/operator-WIDTH.md) or [MASK](../VS_visualcpp/operator-MASK.md) operator was not valid.  
  
 The [WIDTH](../VS_visualcpp/operator-WIDTH.md) operator takes an operand that is the name of a field or a record. The [MASK](../VS_visualcpp/operator-MASK.md) operator takes an operand that is the name of a field or a record type.
