# sincosf Function
Calculates sine and cosine value of _X  
  
## Syntax  
  
```  
inline void sincosf(  
    float _X,  
    _Out_ float * _S,  
    _Out_ float * _C ) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
 `_S`  
 Returns the sine value of _X  
  
 `_C`  
 Returns the cosine value of _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
