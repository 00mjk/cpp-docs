# _bstr_t::Detach
**Microsoft Specific**  
  
 Returns the `BSTR` wrapped by a `_bstr_t` and detaches the `BSTR` from the `_bstr_t`.  
  
## Syntax  
  
```  
  
BSTR Detach( ) throw;  
  
```  
  
## Return Value  
 The `BSTR` wrapped by the `_bstr_t`.  
  
## Example  
 See [_bstr_t::Assign](../VS_visualcpp/_bstr_t--Assign.md) for a example using **Detach**.  
  
 **END Microsoft Specific**
