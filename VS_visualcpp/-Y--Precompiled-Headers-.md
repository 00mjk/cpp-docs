# -Y (Precompiled Headers)
The following compiler options affect the generation and use of precompiled headers:  
  
-   [/Y- (Ignore Precompiled Header Options)](../VS_visualcpp/-Y---Ignore-Precompiled-Header-Options-.md)  
  
-   [/Yc (Create Precompiled Header File)](../VS_visualcpp/-Yc--Create-Precompiled-Header-File-.md)  
  
-   [/Yd (Place Debug Information in Object File)](../VS_visualcpp/-Yd--Place-Debug-Information-in-Object-File-.md)  
  
-   [/Yl (Inject PCH Reference for Debug Library)](../VS_visualcpp/-Yl--Inject-PCH-Reference-for-Debug-Library-.md)  
  
-   [/Yu (Use Precompiled Header File)](../VS_visualcpp/-Yu--Use-Precompiled-Header-File-.md)  
  
 For details on working with precompiled headers, see [Creating Precompiled Header Files](../VS_visualcpp/Creating-Precompiled-Header-Files.md).
