# .Txt Files as Linker Input
LINK expects various text files as additional input. The [command-file](../VS_visualcpp/LINK-Command-Files.md) specifier (@) and the [Base Address](../VS_visualcpp/-BASE--Base-Address-.md) (/BASE), [/DEF](../VS_visualcpp/-DEF--Specify-Module-Definition-File-.md), and [/ORDER](../VS_visualcpp/-ORDER--Put-Functions-in-Order-.md) options all specify text files. These files can have any extension, not just .txt.
