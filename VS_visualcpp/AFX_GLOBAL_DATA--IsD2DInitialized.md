# AFX_GLOBAL_DATA::IsD2DInitialized
Determines whether the D2D was initialized  
  
## Syntax  
  
```  
BOOL IsD2DInitialized() const;  
```  
  
## Return Value  
 TRUE if D2D was initialized; otherwise FALSE.  
  
## Requirements  
 **Header:** afxglobals.h
