# abs Function
Returns the absolute value of the argument  
  
## Syntax  
  
```  
inline int abs(    int _X ) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Integer value  
  
## Return Value  
 Returns the absolute value of the argument.  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency::direct3d
