# Fatal Error C1113
\#using failed on 'file'  
  
 Only a file in the Microsoft Intermediate Language (MSIL) format can be passed to a [#using](../VS_visualcpp/#using-Directive--C---.md) directive. The [/clr](../VS_visualcpp/-clr--Common-Language-Runtime-Compilation-.md) compiler option lets you create an MSIL output file. Other Visual Studio languages also produce MSIL files.
