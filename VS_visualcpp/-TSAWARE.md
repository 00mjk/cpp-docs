# -TSAWARE
```  
/TSAWARE[:NO]  
```  
  
## Remarks  
 The /TSAWARE option to the EDITBIN utility allows you to modify a program image the same way as if you had used the [/TSAWARE](../VS_visualcpp/-TSAWARE--Create-Terminal-Server-Aware-Application-.md) linker option.
