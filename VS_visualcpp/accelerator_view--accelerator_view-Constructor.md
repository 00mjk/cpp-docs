# accelerator_view::accelerator_view Constructor
Initializes a new instance of the [accelerator_view](../VS_visualcpp/accelerator_view-Class.md) class by copying an existing `accelerator_view` object.  
  
## Syntax  
  
```  
accelerator_view(    const accelerator_view & _Other );  
```  
  
#### Parameters  
 `_Other`  
 The `accelerator_view` object to copy.

 **Namespace:** Concurrency
