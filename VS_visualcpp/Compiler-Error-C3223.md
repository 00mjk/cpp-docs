# Compiler Error C3223
'property' : you cannot apply 'typeid' to a property  
  
 You cannot apply [typeid](../VS_visualcpp/typeid---C---Component-Extensions-.md) to a property.  
  
## Example  
 The following sample generates C3223.  
  
```  
// C3223.cpp  
// compile with: /clr  
ref class R {  
public:  
   property int myprop;  
};  
  
int main() {  
   System::Type^ type2 = R::myprop::typeid;   // C3223  
}  
```
