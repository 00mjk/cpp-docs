# outp, outpw
These POSIX functions are deprecated. Use the ISO C++ conformant [_outp, _outpw, _outpd](../VS_visualcpp/_outp--_outpw--_outpd.md) instead.  
  
> [!IMPORTANT]
>  These functions are obsolete. Beginning in Visual Studio 2015, they are not available in the CRT.  
  
> [!IMPORTANT]
>  This API cannot be used in applications that execute in the Windows Runtime. For more information, see [CRT functions not supported with /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
