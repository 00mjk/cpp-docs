# Program Termination
In C++, there are several ways to exit a program:  
  
-   Call the [exit](../VS_visualcpp/exit-Function.md) function.  
  
-   Call the [abort](../VS_visualcpp/abort-Function.md) function.  
  
-   Execute a [return](../VS_visualcpp/return-Statement-in-Program-Termination--C---.md) statement from **main**.
