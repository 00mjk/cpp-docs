# Compiler Warning (level 1) C4805
'operation' : unsafe mix of type 'type' and type 'type' in operation  
  
 This warning is generated for comparison operations between [bool](../VS_visualcpp/bool--C---.md) and [int](../VS_visualcpp/Integer-Types.md). The following sample generates C4805:  
  
```  
// C4805.cpp  
// compile with: /W1  
int main() {  
   int i = 1;  
   bool b = true;  
  
   if (i == b) {   // C4805, comparing bool and int variables  
   }  
}  
```
