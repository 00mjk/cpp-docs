# ML Nonfatal Error A2034
**must be in segment block**  
  
 One of the following was found outside of a segment block:  
  
-   An instruction  
  
-   A label definition  
  
-   A [THIS](../VS_visualcpp/operator-THIS.md) operator  
  
-   A [$](../VS_visualcpp/$.md) operator  
  
-   A procedure definition  
  
-   An [ALIGN](../VS_visualcpp/ALIGN--MASM-.md) directive  
  
-   An [ORG](../VS_visualcpp/ORG.md) directive
