# Dialog Data Exchange and Validation
Dialog data exchange (DDX) is an easy way to initialize the controls in your dialog box and to gather data input by the user. Dialog data validation (DDV) is an easy way to validate data entry in a dialog box. To take advantage of DDX and DDV in your dialog boxes, use the [Add Member Variable Wizard](../VS_visualcpp/Add-Member-Variable-Wizard.md) to create the data members and set their data types and specify validation rules.
