# DB
Allocates and optionally initializes a byte of storage for each `initializer`. `DB` is a synonym of [BYTE](../VS_visualcpp/BYTE--MASM-.md)  
  
## Syntax  
  
```  
[[name]] DB initializer [[, initializer]] ...  
```  
  
## Remarks  
 Can also be used as a type specifier anywhere a type is legal.
