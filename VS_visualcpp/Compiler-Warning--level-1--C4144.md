# Compiler Warning (level 1) C4144
'expression' : relational expression as switch expression  
  
 The specified relational expression was used as the control expression of a [switch](../VS_visualcpp/switch-Statement--C---.md) statement. The associated case statements will be offered Boolean values. The following sample generates C4144:  
  
```  
// C4144.cpp  
// compile with: /W1  
int main()  
{  
   int i = 0;  
   switch(!i) {   // C4144, remove the ! to resolve  
      case 1:  
         break;  
      default:  
         break;  
   }  
}  
```
