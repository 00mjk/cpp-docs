# .Exe Files as Linker Input
The [MS-DOS Stub File Name](../VS_visualcpp/-STUB--MS-DOS-Stub-File-Name-.md) (/STUB) option specifies the name of an .exe file that runs with MS-DOS. LINK examines the specified file to be sure that it is a valid MS-DOS program.
