# Compiler Error C2773
\#import and #using available only in C++ compiler  
  
 The C compiler does not recognize the `#import` preprocessor directive. Compile the source as C++. Use [/TP](../VS_visualcpp/-Tc---Tp---TC---TP--Specify-Source-File-Type-.md) if necessary.
