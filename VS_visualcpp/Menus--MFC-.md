# Menus (MFC)
MFC supplies two elements to help you work with menus:  
  
-   Class [CMenu](../VS_visualcpp/CMenu-Class.md) for manipulating your program's menus at run time. Use the documentation for `CMenu` and the sample to learn how to use `CMenu` effectively.  
  
-   A mechanism for updating menus and toolbar buttons: enabling or disabling them on the fly to suit current program conditions.  
  
 Visual C++ also provides a [menu editor](../VS_visualcpp/Menu-Editor.md) for creating and editing your program's menu resources.
