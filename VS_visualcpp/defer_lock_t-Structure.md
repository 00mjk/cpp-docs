# defer_lock_t Structure
Represents a type that defines a [defer_lock](../VS_visualcpp/-mutex--functions.md#defer_lock_variable) object that is used to select one of the overloaded constructors of [unique_lock](../VS_visualcpp/unique_lock-Class.md).  
  
## Syntax  
  
```cpp  
struct defer_lock_t;  
```
