# STRUCT (MASM)
Declares a structure type having the specified *fielddeclarations*. Each field must be a valid data definition. Same as [STRUC](../VS_visualcpp/STRUC.md).  
  
## Syntax  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
