# CNoMultipleResults Class
Used as a template argument (*TMultiple*) to [CCommand](../VS_visualcpp/CCommand-Class.md) to create an optimized command that handles a single result set.  
  
## Syntax  
  
```  
class CNoMultipleResults  
```  
  
## Remarks  
 If you want a command to handle multiple result sets, use [CMultipleResults](../VS_visualcpp/CMultipleResults-Class.md) instead.  
  
## Requirements  
 **Header:** atldbcli.h
