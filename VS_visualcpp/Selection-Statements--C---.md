# Selection Statements (C++)
The C++ selection statements, [if](../VS_visualcpp/if-else-Statement--C---.md) and [switch](../VS_visualcpp/switch-Statement--C---.md), provide a means to conditionally execute sections of code.  
  
 The [__if_exists](../VS_visualcpp/__if_exists-Statement.md) and [__if_not_exists](../VS_visualcpp/__if_not_exists-Statement.md) statements allow you to conditionally include code depending on the existence of a symbol.  
  
 See the individual topics for the syntax for each statement.
