# Compiler Error C3455
'attribute': none of the attribute constructors matched the arguments  
  
 An invalid value was used to declare an attribute.  See [attribute](../VS_visualcpp/attribute.md) for more information.  
  
## Example  
 The following sample generates C3455.  
  
```  
// C3455.cpp  
// compile with: /clr /c  
using namespace System;  
  
[attribute("MyAt")]   // C3455  
// try the following line instead  
// [attribute(All)]  
ref struct MyAttr {  
   MyAttr() {}  
};  
```
