# ENDS (MASM)
Marks the end of segment, structure, or union *name* previously begun with [SEGMENT](../VS_visualcpp/SEGMENT.md), [STRUCT](../VS_visualcpp/STRUCT--MASM-.md), [UNION](../VS_visualcpp/UNION.md), or a simplified segment directive.  
  
## Syntax  
  
```  
  
name ENDS  
```
