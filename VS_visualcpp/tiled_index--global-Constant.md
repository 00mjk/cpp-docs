# tiled_index::global Constant
Stores an [index](../VS_visualcpp/index-Class.md) object of rank 1, 2, or 3 that represents the global index of an object.  
  
## Syntax  
  
```  
const index<rank> global;  
```  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency
