# Support for ANSI
Most MFC classes and methods support the ANSI character set, although the MFC framework as a whole is gradually evolving toward supporting only the Unicode character set. Because of the ongoing enhancements in Windows Vista and Windows Common Controls version 6.1, support for several ANSI classes and methods is deprecated.  For more information, see [Deprecated ANSI APIs](../VS_visualcpp/Deprecated-ANSI-APIs.md) and [Support for Unicode](../VS_visualcpp/Support-for-Unicode.md).
