# .CONTINUE
Generates code to jump to the top of a [.WHILE](../VS_visualcpp/.WHILE.md) or [.REPEAT](../VS_visualcpp/.REPEAT.md) block if `condition` is true.  
  
## Syntax  
  
```  
  
.CONTINUE [[.IF condition]]  
```
