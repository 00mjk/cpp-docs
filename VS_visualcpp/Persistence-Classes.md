# Persistence Classes
The following classes implement object persistence:  
  
-   [IPersistPropertyBagImpl](../VS_visualcpp/IPersistPropertyBagImpl-Class.md) Allows a client to load and save an object's properties to a property bag.  
  
-   [IPersistStreamInitImpl](../VS_visualcpp/IPersistStreamInitImpl-Class.md) Allows a client to load and save an object's persistent data to a stream.  
  
-   [IPersistStorageImpl](../VS_visualcpp/IPersistStorageImpl-Class.md) Allows a client to load and save an object's persistent data to a storage.
