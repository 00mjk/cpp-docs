# Global Variables and Standard Types
The Microsoft run-time library contains definitions for [global variables](../VS_visualcpp/Global-Variables.md), [control flags](../VS_visualcpp/Control-Flags.md), and [standard types](../VS_visualcpp/Standard-Types.md) used by library routines. Access these variables, flags, and types by declaring them in your program or by including the appropriate header files.
