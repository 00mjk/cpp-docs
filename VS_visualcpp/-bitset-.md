# &lt;bitset&gt;
Defines the template class bitset and two supporting template functions for representing and manipulating fixed-size sequences of bits.  
  
## Syntax  
  
```  
  
#include <bitset>  
  
```  
  
### Operators  
  
|||  
|-|-|  
|[operator&](../VS_visualcpp/-bitset--operators.md#operator_amp_)|Performs a bitwise AND between two bitsets.|  
|[operator<<](../VS_visualcpp/-bitset--operators.md#operator_lt__lt_)|Inserts a text representation of the bit sequence into the standard output stream.|  
|[operator>>](../VS_visualcpp/-bitset--operators.md#operator_gt__gt_)|Inserts a text representation of the bit sequence into the standard input stream.|  
|[operator^](../VS_visualcpp/-bitset--operators.md#operator_xor)|Performs a bitwise EXCLUSIVE-OR between two bitsets.|  
|[operator&#124;](../VS_visualcpp/-bitset--operators.md#operator_or)|Performs a bitwise OR between two bitsets.|  
  
### Classes  
  
|||  
|-|-|  
|[bitset Class](../VS_visualcpp/bitset-Class.md)|The template class describes a type of object that stores a sequence consisting of a fixed number of bits that provide a compact way of keeping flags for a set of items or conditions.|
