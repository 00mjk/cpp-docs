# Linker Tools Error LNK1152
cannot resolve one or more undecorated symbols  
  
 This error is preceded by one warning [LNK4022](../VS_visualcpp/Linker-Tools-Warning-LNK4022.md) for each undecorated symbol that could not be resolved and by at least two warnings [LNK4002](../VS_visualcpp/Linker-Tools-Warning-LNK4002.md) for the duplicate symbols found for the undecorated symbol.
