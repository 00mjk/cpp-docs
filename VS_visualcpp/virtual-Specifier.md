# virtual Specifier
The [virtual](../VS_visualcpp/virtual--C---.md) keyword can be applied only to nonstatic class member functions. It signifies that binding of calls to the function is deferred until run time. For more information, see [Virtual Functions](../VS_visualcpp/Virtual-Functions.md).
