# Contents of a Makefile
A makefile contains:  
  
-   [Description blocks](../VS_visualcpp/Description-Blocks.md)  
  
-   [Commands](../VS_visualcpp/Commands-in-a-Makefile.md)  
  
-   [Macros](../VS_visualcpp/Macros-and-NMAKE.md)  
  
-   [Inference rules](../VS_visualcpp/Inference-Rules.md)  
  
-   [Dot directives](../VS_visualcpp/Dot-Directives.md)  
  
-   [Preprocessing directives](../VS_visualcpp/Makefile-Preprocessing.md)  
  
## Remarks  
 Other features you can use in a makefile are [wildcards](../VS_visualcpp/Wildcards-and-NMAKE.md), [long filenames](../VS_visualcpp/Long-Filenames-in-a-Makefile.md), [comments](../VS_visualcpp/Comments-in-a-Makefile.md), and [special characters](../VS_visualcpp/Special-Characters-in-a-Makefile.md).  
  
 For a sample, see [Sample Makefile](../VS_visualcpp/Sample-Makefile.md).
