# Generic-Text Mappings
To simplify writing code for international markets, generic-text mappings are defined in TCHAR.H for:  
  
-   [Data types](../VS_visualcpp/Data-Type-Mappings.md)  
  
-   [Constants and global variables](../VS_visualcpp/Constant-and-Global-Variable-Mappings.md)  
  
-   [Routine mappings](../VS_visualcpp/Routine-Mappings.md)  
  
 For more information, see [Using Generic-Text Mappings](../VS_visualcpp/Using-Generic-Text-Mappings.md). Generic-text mappings are Microsoft extensions that are not ANSI compatible.
