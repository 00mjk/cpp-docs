# Compiler Error C2669
member function not allowed in anonymous union  
  
 Anonymous unions cannot have member functions.  
  
 The following sample generates C2669:  
  
```  
// C2669.cpp  
struct X {  
   union {  
      int i;  
      void f() {   // C2669, remove function  
         i = 0;   
      }  
   };  
};  
```
