# Compiler Warning (levels 1 and 4) C4223
nonstandard extension used : non-lvalue array converted to pointer  
  
 In standard C, you cannot convert a non-lvalue array to a pointer. With the default Microsoft extensions ([/Ze](../VS_visualcpp/-Za---Ze--Disable-Language-Extensions-.md)), you can.
