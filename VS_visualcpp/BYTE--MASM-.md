# BYTE (MASM)
Allocates and optionally initializes a byte of storage for each `initializer`.  
  
## Syntax  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## Remarks  
 Can also be used as a type specifier anywhere a type is legal.
