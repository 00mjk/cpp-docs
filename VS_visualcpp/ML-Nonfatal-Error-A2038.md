# ML Nonfatal Error A2038
**missing operand for macro operator**  
  
 The assembler found the end of a macro's parameter list immediately after the [!](../VS_visualcpp/operator-!--MASM-Run-Time-.md) or [%](../VS_visualcpp/operator--.md) operator.
