# BSCMAKE Exit Codes
BSCMAKE returns an exit code (also called a return code or error code) to the operating system or the calling program.  
  
|Code|Meaning|  
|----------|-------------|  
|0|No error|  
|1|Command-line error|  
|4|Fatal error during build|
