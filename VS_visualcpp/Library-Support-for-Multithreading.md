# Library Support for Multithreading
All versions of the CRT now support multi threading with the exception of the non-locking versions of some functions. See [Multithreaded Libraries Performance](../VS_visualcpp/Multithreaded-Libraries-Performance.md) for more information.  
  
 See [CRT Library Features](../VS_visualcpp/CRT-Library-Features.md) for more information on CRT versions.
