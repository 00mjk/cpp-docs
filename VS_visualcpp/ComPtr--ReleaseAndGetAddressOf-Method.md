# ComPtr::ReleaseAndGetAddressOf Method
Releases the interface associated with this ComPtr and then retrieves the address of the [ptr_](../VS_visualcpp/ComPtr--ptr_-Data-Member.md) data member, which contains a pointer to the interface that was released.  
  
## Syntax  
  
```  
T** ReleaseAndGetAddressOf();  
```  
  
## Return Value  
 The address of the [ptr_](../VS_visualcpp/ComPtr--ptr_-Data-Member.md) data member of this ComPtr.  
  
## Requirements  
 **Header:** client.h  
  
 **Namespace:** Microsoft::WRL
