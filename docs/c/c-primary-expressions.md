# C Primary Expressions
The operands in expressions are called "primary expressions."  
  
## Syntax  
 *primary-expression*:  
 *identifier*  
  
 *constant*  
  
 *string-literal*  
  
 **(**  *expression*  **)**  
  
 *expression*:  
 *assignment-expression*  
  
 *expression*  **,**  *assignment-expression*
