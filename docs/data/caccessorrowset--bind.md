# CAccessorRowset::Bind
Creates the bindings if you specified **bBind** as false in [CCommand::Open](../data/ccommand--open.md).  
  
## Syntax  
  
```  
  
HRESULT Bind( );  
  
```  
  
## Return Value  
 A standard `HRESULT`.  
  
## Requirements  
 **Header:** atldbcli.h
