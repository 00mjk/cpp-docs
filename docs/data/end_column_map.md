# END_COLUMN_MAP
Marks the end of the column map entries.  
  
## Syntax  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## Remarks  
 It is used with a single accessor on a rowset. The `BEGIN_COLUMN_MAP` macro is completed with the `END_COLUMN_MAP` macro.  
  
## Example  
 See [BEGIN_COLUMN_MAP](../data/begin_column_map.md).  
  
## Requirements  
 **Header:** atldbcli.h
