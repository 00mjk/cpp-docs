# CBookmark::operator =
Assigns a `CBookmark` object to another.  
  
## Syntax  
  
```cpp
      CBookmark& operator =(const CBookmark& bookmark) throw();  
```  
  
## Remarks  
 This operator is needed only in **CBookmark\<0>**.  
  
## Requirements  
 **Header:** atldbcli.h
