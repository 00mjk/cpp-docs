# CBookmark::GetBuffer
Retrieves the pointer to the bookmark buffer.  
  
## Syntax  
  
```  
  
virtual BYTE* GetBuffer( ) const throw( );  
  
```  
  
## Return Value  
 A pointer to the bookmark buffer.  
  
## Requirements  
 **Header:** atldbcli.h
