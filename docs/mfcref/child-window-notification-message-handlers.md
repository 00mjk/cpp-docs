# Child Window Notification Message Handlers
There are five categories of child window notification messages:  
  
|Category|Description|  
|--------------|-----------------|  
|[Generic Control Handler](../mfcref/generic-control-handler.md)|Handler for generic control notification codes.|  
|[User Button Handlers](../mfcref/user-button-handlers.md)|Handlers for user button notification codes.|  
|[Combo Box Handlers](../mfcref/combo-box-handlers.md)|Handlers for combo box notification codes.|  
|[Edit Control Handlers](../mfcref/edit-control-handlers.md)|Handlers for edit control notification codes.|  
|[List Box Handlers](../mfcref/list-box-handlers.md)|Handlers for list box notification codes.|
