# .BREAK
Generates code to terminate a [.WHILE](../intrinsics/.while.md) or [.REPEAT](../intrinsics/.repeat.md) block if *condition* is true.  
  
## Syntax  
  
```  
  
.BREAK [[.IF condition]]   
```
