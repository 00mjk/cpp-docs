# ENDS (MASM)
Marks the end of segment, structure, or union *name* previously begun with [SEGMENT](../intrinsics/segment.md), [STRUCT](../intrinsics/struct--masm-.md), [UNION](../intrinsics/union.md), or a simplified segment directive.  
  
## Syntax  
  
```  
  
name ENDS  
```
