# IFDEF
Grants assembly if *name* is a previously defined label, variable, or symbol.  
  
## Syntax  
  
```  
  
IFDEF name  
```  
  
## Remarks  
 See [IF](../intrinsics/if--masm-.md) for complete syntax.
