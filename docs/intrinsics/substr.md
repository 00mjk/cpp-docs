# SUBSTR
Returns a substring of *textitem*, starting at *position*. The *textitem* can be a literal string, a constant preceded by a `%`, or the string returned by a macro function.  
  
## Syntax  
  
```  
  
name SUBSTR textitem, position [[, length]]  
```
