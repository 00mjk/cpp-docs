# DQ
Allocates and optionally initializes 8 bytes of storage for each `initializer`. Also can be used as a type specifier anywhere a type is legal. `DQ` is a synonym of [QWORD](../intrinsics/qword.md).  
  
## Syntax  
  
```  
[[name]] DQ initializer [[, initializer]]...  
```
