# ML Nonfatal Error A2219
**Bad alignment for offset in unwind code**  
  
 The operand for [.ALLOCSTACK](../intrinsics/.allocstack.md) and [.SAVEREG](../intrinsics/.savereg.md) must be a multiple of 8.  The operand for [.SAVEXMM128](../intrinsics/.savexmm128.md) and [.SETFRAME](../intrinsics/.setframe.md) must be a multiple of 16.
