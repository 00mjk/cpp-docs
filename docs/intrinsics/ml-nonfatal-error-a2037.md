# ML Nonfatal Error A2037
**statement not allowed inside structure definition**  
  
 A structure definition contained an invalid statement.  
  
 A structure cannot contain instructions, labels, procedures, control-flow directives, [.STARTUP](../intrinsics/.startup.md), or [.EXIT](../intrinsics/.exit.md).
