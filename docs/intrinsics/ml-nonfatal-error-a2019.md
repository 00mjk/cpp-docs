# ML Nonfatal Error A2019
**operand must be RECORD type or field**  
  
 The operand following the [WIDTH](../intrinsics/operator-width.md) or [MASK](../intrinsics/operator-mask.md) operator was not valid.  
  
 The [WIDTH](../intrinsics/operator-width.md) operator takes an operand that is the name of a field or a record. The [MASK](../intrinsics/operator-mask.md) operator takes an operand that is the name of a field or a record type.
