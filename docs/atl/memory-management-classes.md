# Memory Management Classes
These classes provide support for heap pointers, smart pointers, and other memory allocation routines.  
  
-   [CAutoPtr](../atl/cautoptr-class.md) This class represents a smart pointer object.  
  
-   [CAutoPtrArray](../atl/cautoptrarray-class.md) This class provides methods useful when constructing an array of smart pointers.  
  
-   [CAutoPtrList](../atl/cautoptrlist-class.md) This class provides methods useful when constructing a list of smart pointers.  
  
-   [CAutoVectorPtr](../atl/cautovectorptr-class.md) This class represents a smart pointer object using vector new and delete operators.  
  
-   [CComAllocator](../atl/ccomallocator-class.md) This class provides methods for managing memory using COM memory routines.  
  
-   [CComGITPtr](../atl/ccomgitptr-class.md) This class provides methods for dealing with interface pointers and the global interface table (GIT).  
  
-   [CComHeap](../atl/ccomheap-class.md) This class implements [IAtlMemMgr](../atl/iatlmemmgr-class.md) using the COM memory allocation functions.  
  
-   [CComHeapPtr](../atl/ccomheapptr-class.md) A smart pointer class for managing heap pointers.  
  
-   [CComPtr](../atl/ccomptr-class.md) A smart pointer class for managing COM interface pointers.  
  
-   [CComPtrBase](../atl/ccomptrbase-class.md) This class provides a basis for smart pointer classes using COM-based memory routines.  
  
-   [CComQIPtr](../atl/ccomqiptr-class.md) A smart pointer class for managing COM interface pointers.  
  
-   [CCRTAllocator](../atl/ccrtallocator-class.md) This class provides methods for managing memory using CRT memory routines.  
  
-   [CCRTHeap](../atl/ccrtheap-class.md) This class implements [IAtlMemMgr](../atl/iatlmemmgr-class.md) using the CRT heap functions.  
  
-   [CGlobalHeap](../atl/cglobalheap-class.md) This class implements [IAtlMemMgr](../atl/iatlmemmgr-class.md) using the Win32 global heap functions.  
  
-   [CHandle](../atl/chandle-class.md) This class provides methods for creating and using a handle object.  
  
-   [CHeapPtr](../atl/cheapptr-class.md) A smart pointer class for managing heap pointers.  
  
-   [CHeapPtrBase](../atl/cheapptrbase-class.md) This class forms the basis for several smart heap pointer classes.  
  
-   [CHeapPtrList](../atl/cheapptrlist-class.md) This class provides methods useful when constructing a list of heap pointers.  
  
-   [CLocalHeap](../atl/clocalheap-class.md) This class implements [IAtlMemMgr](../atl/iatlmemmgr-class.md) using the Win32 local heap functions.  
  
-   [CWin32Heap](../atl/cwin32heap-class.md) This class implements [IAtlMemMgr](../atl/iatlmemmgr-class.md) using the Win32 heap allocation functions.  
  
-   [IAtlMemMgr](../atl/iatlmemmgr-class.md) This class represents the interface to a memory manager.
