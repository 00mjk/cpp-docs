##  <a name="swap"></a>  swap  
 Exchanges the elements of two deques.  
  
```
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```  
  
### Parameters  
 `left`  
 An object of type `deque`.  
  
 `right`  
 An object of type `deque`.  
  
### Example  
 See the example for [deque::swap](../stdcpplib/deque-class.md#deque__swap).
