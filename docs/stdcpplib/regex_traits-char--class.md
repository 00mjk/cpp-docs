# regex_traits&lt;char&gt; Class
Specialization of regex_traits for char.  
  
## Syntax  
  
```  
template <>  
    class regex_traits<char>  
```  
  
## Remarks  
 The class is an explicit specialization of template class [regex_traits Class](../stdcpplib/regex_traits-class.md) for elements of type `char` (so that it can take advantage of library functions that manipulate objects of this type).  
  
## Requirements  
 **Header:** \<regex>  
  
 **Namespace:** std
