# ModuleBase::~ModuleBase Destructor
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
virtual ~ModuleBase();  
```  
  
## Remarks  
 Deinitializes the current instance of the ModuleBase class.
