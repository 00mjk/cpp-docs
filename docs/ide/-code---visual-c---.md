# &lt;code&gt; (Visual C++)
The \<code> tag gives you a way to indicate one or more lines as code.  
  
## Syntax  
  
```  
<code>content</code>  
```  
  
#### Parameters  
 `content`  
 The text you want marked as code.  
  
## Remarks  
 Use [\<c>](../ide/-c---visual-c---.md) to indicate a portion of text should be marked as code.  
  
 Compile with [/doc](../buildref/-doc--process-documentation-comments---c-c---.md) to process documentation comments to a file.
