# GetProjectPath
Returns the project's directory path.  
  
## Syntax  
  
```  
  
      function GetProjectPath(   
   oProj    
);  
```  
  
#### Parameters  
 `oProj`  
 The selected project  
  
## Return Value  
 The project's directory path.  
  
## Remarks  
 Call this function to get the project's path.
