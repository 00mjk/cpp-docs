# Environmental Constants
## Syntax  
  
```  
#include <stdlib.h>  
```  
  
## Remarks  
 This constant defines the environmental length for strings.  
  
|Constant|Meaning|  
|--------------|-------------|  
|`_MAX_ENV`|Maximum string size of an environmental string.|
