# BUFSIZ
## Syntax  
  
```  
  
#include <stdio.h>  
```  
  
## Remarks  
 `BUFSIZ` is the required user-allocated buffer for the [setvbuf](../c-runtime-library/reference/setvbuf.md) routine.
