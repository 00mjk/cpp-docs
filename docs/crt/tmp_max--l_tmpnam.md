# TMP_MAX, L_tmpnam
## Syntax  
  
```  
#include <stdio.h>  
```  
  
## Remarks  
 `TMP_MAX` is the maximum number of unique filenames that the `tmpnam` function can generate. `L_tmpnam` is the length of temporary filenames generated by `tmpnam`.
