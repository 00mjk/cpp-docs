# Global Variables and Standard Types
The Microsoft run-time library contains definitions for [global variables](../crt/global-variables.md), [control flags](../crt/control-flags.md), and [standard types](../crt/standard-types.md) used by library routines. Access these variables, flags, and types by declaring them in your program or by including the appropriate header files.
