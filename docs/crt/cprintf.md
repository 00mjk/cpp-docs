# cprintf
This POSIX function is deprecated. Use the ISO C++ conformant [_cprintf](../crt/_cprintf--_cprintf_l--_cwprintf--_cwprintf_l.md) or security-enhanced [_cprintf_s](../crt/_cprintf_s--_cprintf_s_l--_cwprintf_s--_cwprintf_s_l.md) instead.  
  
> [!IMPORTANT]
>  This API cannot be used in applications that execute in the Windows Runtime. For more information, see [CRT functions not supported with /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
