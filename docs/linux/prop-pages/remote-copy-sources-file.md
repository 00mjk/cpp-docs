# Remote Copy Sources Properties (Linux C++)
The properties set on this property page apply to the selected file.

Property | Description
--- | ---
Remote File | Specifies where to copy the file on the remote system. The option overrides the default name and location of the file on the remote system. 
Copy Remotely | Specifies whether to copy the file on the remote system.
Copy To Output Directory | Specifies whether to also copy the file to the output project directory on the remote system.
