# Selection Statements (C++)
The C++ selection statements, [if](../cpp/if-else-statement--c---.md) and [switch](../cpp/switch-statement--c---.md), provide a means to conditionally execute sections of code.  
  
 The [__if_exists](../cpp/__if_exists-statement.md) and [__if_not_exists](../cpp/__if_not_exists-statement.md) statements allow you to conditionally include code depending on the existence of a symbol.  
  
 See the individual topics for the syntax for each statement.
