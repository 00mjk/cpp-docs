# utility (STL/CLR)
Include the STL/CLR header `<cliext/utility>` to define the template class `pair` and several supporting template functions.  
  
## Syntax  
  
```  
#include <utility>  
```  
  
## Declarations  
  
|Class|Description|  
|-----------|-----------------|  
|[pair (STL/CLR)](../cli/pair--stl-clr-.md)|Wrap a pair of elements.|  
  
|Operator|Description|  
|--------------|-----------------|  
|[operator== (pair) (STL/CLR)](../cli/operator==--pair---stl-clr-.md)|Pair equal comparison.|  
|[operator!= (pair) (STL/CLR)](../cli/operator!=--pair---stl-clr-.md)|Pair not equal comparison.|  
|[operator< (pair) (STL/CLR)](../cli/operator---pair---stl-clr-.md)|Pair less than comparison.|  
|[operator\<= (pair) (STL/CLR)](../cli/operator-=--pair---stl-clr-.md)|Pair less than or equal comparison.|  
|[operator> (pair) (STL/CLR)](../cli/operator---pair---stl-clr-.md)|Pair greater than comparison.|  
|[operator>= (pair) (STL/CLR)](../cli/operator-=--pair---stl-clr-.md)|Pair greater than or equal comparison.|  
  
|Function|Description|  
|--------------|-----------------|  
|[make_pair (STL/CLR)](../cli/make_pair--stl-clr-.md)|Make a pair from a pair of values.|  
  
## Requirements  
 **Header:** \<cliext/utility>  
  
 **Namespace:** cliext
