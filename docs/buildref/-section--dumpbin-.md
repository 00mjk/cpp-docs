# /SECTION (DUMPBIN)
```  
/SECTION:section  
```  
  
## Remarks  
 This option restricts the output to information on the specified *section*. Use the [/HEADERS](../buildref/-headers.md) option to get a list of sections in the file.  
  
 Only the [/HEADERS](../buildref/-headers.md) DUMPBIN option is available for use on files produced with the [/GL](../buildref/-gl--whole-program-optimization-.md) compiler option.
