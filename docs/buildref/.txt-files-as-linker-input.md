# .Txt Files as Linker Input
LINK expects various text files as additional input. The [command-file](../buildref/link-command-files.md) specifier (@) and the [Base Address](../buildref/-base--base-address-.md) (/BASE), [/DEF](../buildref/-def--specify-module-definition-file-.md), and [/ORDER](../buildref/-order--put-functions-in-order-.md) options all specify text files. These files can have any extension, not just .txt.
