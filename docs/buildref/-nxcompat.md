# /NXCOMPAT
```  
/NXCOMPAT[:NO]  
```  
  
## Remarks  
 Indicates that an executable was tested to be compatible with the Windows Data Execution Prevention feature.  
  
 For more information, see [/NXCOMPAT (Compatible with Data Execution Prevention)](../buildref/-nxcompat--compatible-with-data-execution-prevention-.md).
