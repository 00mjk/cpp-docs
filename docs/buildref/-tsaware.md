# /TSAWARE
```  
/TSAWARE[:NO]  
```  
  
## Remarks  
 The /TSAWARE option to the EDITBIN utility allows you to modify a program image the same way as if you had used the [/TSAWARE](../buildref/-tsaware--create-terminal-server-aware-application-.md) linker option.
