# Windows Sockets
This family of articles covers the MFC implementation of Windows Sockets. MFC supplies two classes to support programming network applications with the Windows Sockets API. Class [CAsyncSocket](../mfcref/casyncsocket-class.md) encapsulates the Windows Sockets API one for one, giving advanced network programmers the most power and flexibility. Class [CSocket](../mfcref/csocket-class.md) provides a simplified interface for serializing data to and from a [CArchive](../mfcref/carchive-class.md) object.
