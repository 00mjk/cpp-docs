# DWORD

Allocates and optionally initializes a double word (4 bytes) of storage for each *initializer*. **DWORD** is a synonym of [DD](../../assembler/masm/dd.md).

## Syntax

> ⟦*name*⟧ **DWORD** *initializer* ⟦__,__ *initializer* ...⟧

## Remarks

Can also be used as a type specifier anywhere a type is legal.
