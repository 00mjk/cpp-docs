# .BREAK (32-bit MASM)

Generates code to terminate a [.WHILE](../../assembler/masm/dot-while.md) or [.REPEAT](../../assembler/masm/dot-repeat.md) block if *condition* is true. (32-bit MASM only.)

## Syntax

> .BREAK [[.IF condition]]
