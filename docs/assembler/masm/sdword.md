# SDWORD
Allocates and optionally initializes a signed double word (4 bytes) of storage for each `initializer`. Also can be used as a type specifier anywhere a type is legal.  
  
## Syntax  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
