# IFIDN, IFIDNI

Grants assembly if the text items are identical.

## Syntax

> **IFIDN** [[**I**]] *textitem1*__,__ *textitem2*

## Remarks

If **IFIDNI** is given, the comparison is case insensitive. See [IF](../../assembler/masm/if-masm.md) for complete syntax.
