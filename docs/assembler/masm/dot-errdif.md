# &period;ERRDIF, &period;ERRDIFI

Generates an error if the text items are different.

## Syntax

> **.ERRDIF**\[**I**] *textitem1*__,__ *textitem2* \[__,__ *message*]

## Remarks

If **&period;ERRDIFI** is given, the comparison is case insensitive.
