# ELSEIF (MASM)

Combines [ELSE](../../assembler/masm/else-masm.md) and [IF](../../assembler/masm/if-masm.md) into one statement.

## Syntax

> **ELSEIF** *constantExpression*
> *statements*
> ⟦**ELSE**\
> *else-statements*⟧\
> **ENDIF**

## Remarks

See [IF](../../assembler/masm/if-masm.md) for more information.
