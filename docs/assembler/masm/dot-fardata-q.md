# .FARDATA? (32-bit MASM)

When used with [.MODEL](../../assembler/masm/dot-model.md), starts a far data segment for uninitialized data (segment name FAR_BSS or *name*). (32-bit MASM only.)

## Syntax

> **.FARDATA?** ⟦*name*⟧
