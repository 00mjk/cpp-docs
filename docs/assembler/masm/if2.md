# IF2
**IF** block is evaluated on every assembly pass if **OPTION:SETIF2** is **TRUE**.  
  
## Syntax  
  
```  
  
IF2   
expression  
  
```  
  
## Remarks  
 See [IF](../../assembler/masm/if-masm.md) for complete syntax.
