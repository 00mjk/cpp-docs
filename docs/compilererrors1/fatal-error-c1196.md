# Fatal Error C1196
'identifier' : identifier found in type library 'typelib' is not a valid C++ identifier  
  
 One of the identifiers in your type library is not a valid C++ identifier. The type library is not available for use with [#import](../c/sharpimport-directive--c---.md).
