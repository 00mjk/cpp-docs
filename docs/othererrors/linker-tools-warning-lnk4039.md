# Linker Tools Warning LNK4039
section 'name' specified with /SECTION option does not exist  
  
 [DUMPBIN](../buildref/dumpbin-reference.md) or [EDITBIN](../buildref/editbin-reference.md) could not find a section called `name` in the input file.  
  
 To see the sections in a file, run DUMPBIN with the [/HEADERS](../buildref/-headers.md) option.
