# StringReference::GetHSTRING Method
Returns the contents of the string as an `__abi_HSTRING`.  
  
## Syntax  
  
```scr  
__abi_HSTRING GetHSTRING() const  
  
```  
  
## Return Value  
 An `__abi_HSTRING` that contains the string data.
