# Vector::Append Method
Inserts the specified item after the last item in the current Vector.  
  
## Syntax  
  
```  
  
virtual void Append(  
   T item  
);  
```  
  
#### Parameters  
 `index`  
 The item to insert into the Vector. The type of `item` is defined by the *T* typename.  
  
## Requirements  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
