# default::(type_name)::Equals Method
Determines whether the specified object is equal to the current object.  
  
## Syntax  
  
```cpp  
  
bool Equals(  
    Object^ obj  
)  
```  
  
## Parameters  
 obj  
 The object to compare.  
  
## Return Value  
 `true` if the objects are equal, otherwise `false`.
