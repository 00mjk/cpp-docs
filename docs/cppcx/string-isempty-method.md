# String::IsEmpty Method
Indicates whether the current String object is empty.  
  
## Syntax  
  
```cpp  
  
bool IsEmpty()  
```  
  
## Return Value  
 `true` if the current String object is `null` or the empty string (L""); otherwise, `false`.
