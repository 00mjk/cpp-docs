# Vector::GetView Method
Returns a read-only view of a Vector; that is, an IVectorView.  
  
## Syntax  
  
```  
  
Windows::Foundation::Collections::IVectorView<T>^   
   GetView();  
```  
  
## Return Value  
 An IVectorView object.  
  
## Requirements  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
