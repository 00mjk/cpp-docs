# Vector::Size Method
Returns the number of elements in the current Vector object.  
  
## Syntax  
  
```  
  
virtual property unsigned int Size;  
```  
  
## Return Value  
 The number of elements in the current Vector.  
  
## Requirements  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
