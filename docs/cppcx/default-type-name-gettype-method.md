# default::(type_name)::GetType Method
Returns a Platform::Type^ that represents the current type.  
  
## Syntax  
  
```cpp  
  
Platform::Type^ GetType()  
```  
  
### Return Value  
 A [Platform::Type](../cppcx/platform-type-class.md)^ object that represents the current object.
