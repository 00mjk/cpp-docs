# UnorderedMap::Remove Method
Deletes the specified key-value pair from the UnorderedMap object.  
  
## Syntax  
  
```cpp  
  
virtual void Remove(  
   K key);  
```  
  
#### Parameters  
 `key`  
 The key portion of the key-value pair. The type of `key` is typename *K*.  
  
## Requirements  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
