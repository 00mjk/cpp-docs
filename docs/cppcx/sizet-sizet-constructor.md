# SizeT::SizeT constructor
Initializes a new instance of SizeT with the specified value.  
  
## Syntax  
  
```cpp  
SizeT( uint32 value1 );   SizeT( void* value2 );  
```  
  
## Parameters  
 value1  
 An unsigned 32-bit value.  
  
 value2  
 Pointer to an unsigned 32-bit value.
