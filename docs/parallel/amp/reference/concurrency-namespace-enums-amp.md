##  <a name="access_type_enumeration"></a>  access_type Enumeration

```  
enum access_type;  
```  
  
##  <a name="queuing_mode_enumeration"></a>  queuing_mode Enumeration  
 Specifies the queuing modes that are supported on the accelerator.  
  
```  
enum queuing_mode;  
```
