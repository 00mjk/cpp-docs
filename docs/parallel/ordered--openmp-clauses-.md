# ordered (OpenMP Clauses)
Required on a parallel [for](../parallel/for--openmp-.md) statement if an [ordered](../parallel/ordered--openmp-directives-.md) directive is to be used in the loop.  
  
## Syntax  
  
```  
ordered  
```  
  
## Remarks  
 `ordered` applies to the [for](../parallel/for--openmp-.md) directive.  
  
 For more information, see [2.4.1 for Construct](../parallel/2.4.1-for-construct.md).  
  
## Example  
 See [ordered](../parallel/ordered--openmp-directives-.md) for an example of using `ordered` clause.
