# _bstr_t::length
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [_bstr_t::length](https://docs.microsoft.com/cpp/cpp/bstr-t-length).  
  
Microsoft Specific**  
  
 Returns the number of characters in the `_bstr_t`, not including the terminating null, of the encapsulated `BSTR`.  
  
## Syntax  
  
```  
  
unsigned int length ( ) const throw( );  
  
```
