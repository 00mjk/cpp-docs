# Jump Statements (C++)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Jump Statements (C++)](https://docs.microsoft.com/cpp/cpp/jump-statements-cpp).  
  
A C++ jump statement performs an immediate local transfer of control.  
  
## Syntax  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
