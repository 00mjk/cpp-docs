# _bstr_t::operator !
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [_bstr_t::operator !](https://docs.microsoft.com/cpp/cpp/bstr-t-operator-logical-not).  
  
Microsoft Specific**  
  
 Checks if the encapsulated `BSTR` is a **NULL** string.  
  
## Syntax  
  
```  
  
bool operator!( ) const throw( );  
  
```  
  
## Return Value  
 It returns **true** if yes, **false** if not.  
  
 **END Microsoft Specific**
