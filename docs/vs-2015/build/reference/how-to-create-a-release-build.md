# How to: Create a Release Build
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [How to: Create a Release Build](https://docs.microsoft.com/cpp/build/reference/how-to-create-a-release-build).  
  
  
To generate a release build of your program  
  
1.  Select **Release** from the **Solution Configuration** drop-down list, which is on the **Standard** toolbar.  
  
2.  On the **Build** menu, click **Build**.
