# Improving Compiler Throughput
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Improving Compiler Throughput](https://docs.microsoft.com/cpp/build/reference/improving-compiler-throughput).  
  
  
Use precompiled header files to build your project faster. This is important if you are using ATL, MFC, or the [!INCLUDE[winsdkshort](../../includes/winsdkshort-md.md)] header files.  
  
 See [/Yc](../../build/reference/yc-create-precompiled-header-file.md) and [/Yu](../../build/reference/yu-use-precompiled-header-file.md).  
  
 For more information on precompiled headers, see [Creating Precompiled Header Files](../../build/reference/creating-precompiled-header-files.md).
