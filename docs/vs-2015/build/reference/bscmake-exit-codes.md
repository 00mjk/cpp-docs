# BSCMAKE Exit Codes
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [BSCMAKE Exit Codes](https://docs.microsoft.com/cpp/build/reference/bscmake-exit-codes).  
  
  
BSCMAKE returns an exit code (also called a return code or error code) to the operating system or the calling program.  
  
|Code|Meaning|  
|----------|-------------|  
|0|No error|  
|1|Command-line error|  
|4|Fatal error during build|
