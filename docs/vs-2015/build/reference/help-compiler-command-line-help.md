# /HELP (Compiler Command-Line Help)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [-HELP (Compiler Command-Line Help)](https://docs.microsoft.com/cpp/build/reference/help-compiler-command-line-help).  
  
  
Displays a listing of compiler options to standard output.  
  
## Syntax  
  
```  
/HELP  
/help  
/?  
```  
  
## Remarks  
  
### To set this compiler option in the Visual Studio development environment  
  
-   This compiler option should only be accessed from the command line.  
  
### To set this compiler option programmatically  
  
-   This compiler option cannot be changed programmatically.
