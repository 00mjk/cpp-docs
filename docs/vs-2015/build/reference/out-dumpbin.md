# /OUT (DUMPBIN)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [-OUT (DUMPBIN)](https://docs.microsoft.com/cpp/build/reference/out-dumpbin).  
  
  
OUT:filename  
```  
  
## Remarks  
 This option specifies a *filename* for the output. By default, DUMPBIN displays the information to standard output.  
  
 Only the [/HEADERS](../../build/reference/headers.md) DUMPBIN option is available for use on files produced with the [/GL](../../build/reference/gl-whole-program-optimization.md) compiler option.
