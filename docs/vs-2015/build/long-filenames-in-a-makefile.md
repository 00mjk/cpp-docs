# Long Filenames in a Makefile
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Long Filenames in a Makefile](https://docs.microsoft.com/cpp/build/long-filenames-in-a-makefile).  
  
  
Enclose long filenames in double quotation marks, as follows:  
  
```  
all : "VeryLongFileName.exe"  
```
