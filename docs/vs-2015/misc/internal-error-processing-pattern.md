# Internal error processing pattern.
This error generally occurs when a regular expression has been included in a search string but the syntax for the regular expression contains errors.  
  
### To correct this error  
  
1.  Re-enter the search string using the correct regular expression syntax. For more information, see [Regular Expressions](http://msdn.microsoft.com/library/718a617d-0e05-47e1-a218-9746971527f4).
