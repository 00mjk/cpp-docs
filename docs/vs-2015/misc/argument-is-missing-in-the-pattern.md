# Argument is missing in the pattern.
This error generally occurs when incomplete syntax for a regular expression has been entered in the search string.  
  
### To correct this error  
  
1.  Enter the string again. For more information on syntax, see [Regular Expressions](http://msdn.microsoft.com/library/718a617d-0e05-47e1-a218-9746971527f4).
