# &lt;cmath&gt;
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [&lt;cmath&gt;](https://docs.microsoft.com/cpp/standard-library/cmath).  
  
Includes the Standard C library header \<math.h> and adds the associated names to the `std` namespace.  
  
## Syntax  
  
```  
#include <cmath>  
  
```  
  
## Remarks  
 Including this header ensures that the names declared using external linkage in the Standard C library header are declared in the `std` namespace.
