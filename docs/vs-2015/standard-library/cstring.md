# &lt;cstring&gt;
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [&lt;cstring&gt;](https://docs.microsoft.com/cpp/standard-library/cstring).  
  
Includes the Standard C library header \<string.h> and adds the associated names to the `std` namespace.  
  
## Syntax  
  
```  
#include <cstring>  
  
```  
  
## Remarks  
 Including this header ensures that the names declared using external linkage in the Standard C library header are declared in the `std` namespace.
