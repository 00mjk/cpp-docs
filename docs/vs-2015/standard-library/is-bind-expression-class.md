# is_bind_expression Class
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [is_bind_expression Class](https://docs.microsoft.com/cpp/standard-library/is-bind-expression-class).  
  
Tests if type generated by calling `bind`.  
  
## Syntax  
  
struct is_bind_expression {  
   static const bool value;  
   };  
  
## Remarks  
 The constant value `value` is true if the type `Ty` is a type returned by a call to `bind`, otherwise false.  
  
## Example  
  
```  
// std_tr1__functional__is_bind_expression.cpp   
// compile with: /EHsc   
#include <functional>   
#include <iostream>   
  
void square(double x)   
    {   
    std::cout << x << "^2 == " << x * x << std::endl;   
    }   
  
template<class Expr>   
    void test_for_bind(const Expr&)   
    {   
    std::cout << std::is_bind_expression<Expr>::value << std::endl;   
    }   
  
int main()   
    {   
    test_for_bind(3.0 * 3.0);   
    test_for_bind(std::bind(square, 3));   
  
    return (0);   
    }  
  
```  
  
```Output  
0  
1  
```
