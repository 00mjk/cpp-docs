The latest version of this topic can be found at [bad_function_call Class](https://docs.microsoft.com/cpp/standard-library/bad-function-call-class).

## Syntax  
  
```  
class bad_function_call  
 : public std::exception {  
 };  
```  
  
## Remarks  
 The class describes an exception thrown to indicate that a call to `operator()` on a [function Class](../standard-library/function-class.md)
