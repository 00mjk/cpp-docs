# &lt;cerrno&gt;
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [&lt;cerrno&gt;](https://docs.microsoft.com/cpp/standard-library/cerrno).  
  
Includes the Standard C library header \<errno.h> and adds the associated names to the `std` namespace.  
  
## Syntax  
  
```  
#include <cerrno>  
  
```  
  
## Remarks  
 Including this header ensures that the names declared using external linkage in the Standard C library header are declared in the `std` namespace.
