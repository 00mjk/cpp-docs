# Container Class::begin
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Container Class::begin](https://docs.microsoft.com/cpp/standard-library/container-class-begin).  
  
NOTE]
>  This topic is in the Visual C++ documentation as a nonfunctional example of containers used in the Standard C++ Library. For more information, see [STL Containers](../standard-library/stl-containers.md).  
  
 Returns an iterator that points at the first element of the sequence (or just beyond the end of an empty sequence).  
  
## Syntax  
  
```  
 
    const_iterator begin() const;

 
iterator begin();
```
