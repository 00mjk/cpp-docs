# &lt;ctime&gt;
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [&lt;ctime&gt;](https://docs.microsoft.com/cpp/standard-library/ctime).  
  
Includes the Standard C library header \<time.h> and adds the associated names to the `std` namespace.  
  
## Syntax  
  
```  
#include <ctime>  
  
```  
  
## Remarks  
 Including this header ensures that the names declared using external linkage in the Standard C library header are declared in the `std` namespace.
