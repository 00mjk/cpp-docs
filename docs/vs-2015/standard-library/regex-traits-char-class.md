# regex_traits&lt;char&gt; Class
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [regex_traits&lt;char&gt; Class](https://docs.microsoft.com/cpp/standard-library/regex-traits-char-class).  
  
Specialization of regex_traits for char.  
  
## Syntax  
  
```  
template <>  
class regex_traits<char>  
```  
  
## Remarks  
 The class is an explicit specialization of template class [regex_traits Class](../standard-library/regex-traits-class.md) for elements of type `char` (so that it can take advantage of library functions that manipulate objects of this type).  
  
## Requirements  
 **Header:** \<regex>  
  
 **Namespace:** std
