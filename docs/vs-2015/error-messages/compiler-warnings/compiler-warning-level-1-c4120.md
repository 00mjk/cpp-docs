# Compiler Warning (level 1) C4120
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Warning (level 1) C4120](https://docs.microsoft.com/cpp/error-messages/compiler-warnings/compiler-warning-level-1-c4120).  
  
  
based/unbased mismatch  
  
 The compiler cannot convert between the two pointers because one is based and the other is not.
