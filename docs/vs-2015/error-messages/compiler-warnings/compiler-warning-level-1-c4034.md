# Compiler Warning (level 1) C4034
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Warning (level 1) C4034](https://docs.microsoft.com/cpp/error-messages/compiler-warnings/compiler-warning-level-1-c4034).  
  
  
sizeof returns 0  
  
 The `sizeof` operator is applied to an operand of size zero (an empty structure, union, class, or enumerated type, or type `void`).
