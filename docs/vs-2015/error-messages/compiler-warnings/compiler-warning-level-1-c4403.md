# Compiler Warning (level 1) C4403
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Warning (level 1) C4403](https://docs.microsoft.com/cpp/error-messages/compiler-warnings/compiler-warning-level-1-c4403).  
  
  
illegal PTR operator  
  
 A PTR operator is used inappropriately in inline assembler code.  
  
 The compiler ignored the PTR operator.
