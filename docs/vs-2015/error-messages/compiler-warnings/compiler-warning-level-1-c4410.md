# Compiler Warning (level 1) C4410
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Warning (level 1) C4410](https://docs.microsoft.com/cpp/error-messages/compiler-warnings/compiler-warning-level-1-c4410).  
  
  
illegal size for operand  
  
 One of the operands on the instruction had an incorrect size. The smallest legal size for the operand was used.
