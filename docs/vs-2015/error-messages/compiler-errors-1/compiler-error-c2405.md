# Compiler Error C2405
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2405](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2405).  
  
  
illegal short forward reference with offset  
  
 Short forward references must refer to a label only. An additional offset cannot be used.
