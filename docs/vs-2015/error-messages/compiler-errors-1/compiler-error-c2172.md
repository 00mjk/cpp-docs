# Compiler Error C2172
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2172](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2172).  
  
  
function' : actual parameter is not a pointer : parameter number  
  
 Parameter `number` is not a pointer. The function expects a pointer.
