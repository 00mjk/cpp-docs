# Fatal Error C1064
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Fatal Error C1064](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/fatal-error-c1064).  
  
  
compiler limit : token overflowed internal buffer  
  
 An identifier exceeds the length of the internal buffer used for identifiers. Shorten the name.
