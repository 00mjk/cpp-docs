# Compiler Error C2417
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2417](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2417).  
  
  
divide by zero in 'context'  
  
 The parameter to the right of the division operator is zero in this context.
