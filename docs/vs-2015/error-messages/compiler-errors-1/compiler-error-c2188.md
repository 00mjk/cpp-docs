# Compiler Error C2188
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2188](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2188).  
  
  
number' : too big for wide character  
  
 The number exceeds the size limit for the wide-character type. Choose a larger type.
