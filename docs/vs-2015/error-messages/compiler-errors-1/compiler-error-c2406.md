# Compiler Error C2406
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2406](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2406).  
  
  
identifier' : name undefined in 'context'  
  
 An undefined identifier is used with the `SIZE`, `LENGTH`, or member-selection (.) operator.
