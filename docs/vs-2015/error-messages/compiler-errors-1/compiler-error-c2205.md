# Compiler Error C2205

The latest version of this topic can be found at [Compiler Error C2205](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2205).  
  
  
identifier' : cannot initialize extern variables with block scope  
  
 An `extern` variable cannot be initialized in a function.
