# Compiler Error C2169
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2169](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2169).  
  
  
function' : intrinsic function, cannot be defined  
  
 A function definition appears for a function already declared `intrinsic`.
