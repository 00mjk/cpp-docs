# Compiler Error C2295
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2295](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2295).  
  
  
escaped 'character' : is illegal in macro definition  
  
 A macro definition cannot contain an escape sequence with the specified character.
