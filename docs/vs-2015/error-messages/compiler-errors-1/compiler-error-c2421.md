# Compiler Error C2421
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2421](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2421).  
  
  
PTR operator used with register in 'context'  
  
 The `PTR` operator cannot be used with a `register` operand.
