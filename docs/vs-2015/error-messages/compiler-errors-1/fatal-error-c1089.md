# Fatal Error C1089
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Fatal Error C1089](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/fatal-error-c1089).  
  
  
Cannot truncate filetype file: 'file': message  
  
 The compiler cannot shrink a file to zero length.
