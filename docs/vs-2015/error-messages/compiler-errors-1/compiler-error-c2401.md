# Compiler Error C2401
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2401](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2401).  
  
  
identifier' : register must be base in 'context'  
  
 The register used in an indirect memory operand must be a base register in this context.
