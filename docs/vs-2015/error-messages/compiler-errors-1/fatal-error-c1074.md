# Fatal Error C1074
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Fatal Error C1074](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/fatal-error-c1074).  
  
  
IDB' is illegal extension for PDB file: filename  
  
 The compiler expects program databases to have the .pdb extension.
