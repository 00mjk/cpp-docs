# Fatal Error C1065
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Fatal Error C1065](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/fatal-error-c1065).  
  
  
compiler limit : out of tags  
  
 The source file contains more than 65,523 classes, structs, unions, namespaces, or enums.  
  
 Split the file into smaller files.
