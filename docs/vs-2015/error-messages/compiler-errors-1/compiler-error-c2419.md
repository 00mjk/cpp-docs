# Compiler Error C2419
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2419](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2419).  
  
  
mod by zero in 'context'  
  
 The parameter to the right of the `MOD` operator is zero in this context.
