# Compiler Error C2320
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2320](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2320).  
  
  
expected ':' to follow access specifier 'specifier'  
  
 The keyword `public`, `protected`, or `private` must be followed by a colon.
