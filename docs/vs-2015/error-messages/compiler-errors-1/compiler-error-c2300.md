# Compiler Error C2300
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2300](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2300).  
  
  
identifier' : class does not have a destructor called '~identifier'  
  
 The class does not have a destructor with the required name.
