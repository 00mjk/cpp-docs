The latest version of this topic can be found at [Compiler Error C2242](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2242).  
  
  
typedef name cannot follow class/struct/union  
  
 A `typedef` name appears at the end of a qualified name.
