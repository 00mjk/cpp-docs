# Compiler Error C2200
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2200](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2200).  
  
  
function' : function has already been defined  
  
 An `alloc_text` pragma uses a function name already defined.
