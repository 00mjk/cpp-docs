# Compiler Error C2418
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2418](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2418).  
  
  
cannot delete browser file: filename  
  
 The compiler could not delete the browser file.  
  
### To fix by checking the following possible causes  
  
1.  File is read-only.  
  
2.  File is in use by another process.
