# Fatal Error C1003
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Fatal Error C1003](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/fatal-error-c1003).  
  
  
error count exceeds number; stopping compilation  
  
 Errors in the program are too numerous to allow recovery. The compiler must terminate.
