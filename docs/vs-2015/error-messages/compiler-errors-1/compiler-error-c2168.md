# Compiler Error C2168
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2168](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2168).  
  
  
function' : too few actual parameters for intrinsic function  
  
 A reference to an `intrinsic` function has too few parameters.
