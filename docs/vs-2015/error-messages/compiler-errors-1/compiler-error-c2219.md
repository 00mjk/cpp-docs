# Compiler Error C2219
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2219](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2219).  
  
  
syntax error : type qualifier must be after '*'  
  
 Type qualifier (`const` or `volatile`) appears where it is not permitted.
