# Compiler Error C2028
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2028](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2028).  
  
  
struct/union member must be inside a struct/union  
  
 Structure or union members must be declared within the structure or union.
