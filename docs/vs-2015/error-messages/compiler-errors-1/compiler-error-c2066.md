# Compiler Error C2066
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2066](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2066).  
  
  
cast to function type is illegal  
  
 In ANSI C, it is not legal to cast between a function pointer and a data pointer.
