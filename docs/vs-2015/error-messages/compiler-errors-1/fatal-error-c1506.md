# Fatal Error C1506
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Fatal Error C1506](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/fatal-error-c1506).  
  
  
unrecoverable block scoping error  
  
 The block was too large to compile.  
  
### To fix by checking the following possible causes  
  
1.  Mismatched braces  
  
2.  Unusually large function or class
