# Compiler Error C2413
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2413](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2413).  
  
  
token' : illegal align size  
  
 The size used with the `ALIGN` directive is missing or outside the valid range.
