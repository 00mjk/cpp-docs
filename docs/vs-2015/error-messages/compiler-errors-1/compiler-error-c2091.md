# Compiler Error C2091
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2091](https://docs.microsoft.com/cpp/error-messages/compiler-errors-1/compiler-error-c2091).  
  
  
function returns function  
  
 A function cannot return a function. Return a pointer to a function instead.
