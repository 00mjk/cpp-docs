# Expression Evaluator Error CXX0045
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Expression Evaluator Error CXX0045](https://docs.microsoft.com/cpp/error-messages/tool-errors/expression-evaluator-error-cxx0045).  
  
  
not a function  
  
 An argument list was supplied for a symbol in the program that is not the name of a function.  
  
## Example  
  
```  
queue( alpha, beta )  
```  
  
 when `queue` is not a function.  
  
 This error is identical to CAN0045.
