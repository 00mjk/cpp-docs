# Resource Compiler Fatal Error RC1121
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Resource Compiler Fatal Error RC1121](https://docs.microsoft.com/cpp/error-messages/tool-errors/resource-compiler-fatal-error-rc1121).  
  
  
I/O error reading file  
  
 The Resource Compiler was not able to read a file. Check that the drive containing the file is available and that the file is valid.
