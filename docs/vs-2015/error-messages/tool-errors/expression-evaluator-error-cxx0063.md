# Expression Evaluator Error CXX0063
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Expression Evaluator Error CXX0063](https://docs.microsoft.com/cpp/error-messages/tool-errors/expression-evaluator-error-cxx0063).  
  
  
overloaded operator -> not supported  
  
 The expression used an overloaded class member access operator (**->**).  
  
 This error is identical to CAN0063.
