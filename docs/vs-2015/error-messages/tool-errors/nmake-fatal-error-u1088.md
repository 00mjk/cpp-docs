# NMAKE Fatal Error U1088
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [NMAKE Fatal Error U1088](https://docs.microsoft.com/cpp/error-messages/tool-errors/nmake-fatal-error-u1088).  
  
  
invalid separator '::' on inference rule  
  
 An inference rule must be followed by a single colon (**:**).
