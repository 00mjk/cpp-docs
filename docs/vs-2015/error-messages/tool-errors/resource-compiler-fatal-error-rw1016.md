# Resource Compiler Fatal Error RW1016
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Resource Compiler Fatal Error RW1016](https://docs.microsoft.com/cpp/error-messages/tool-errors/resource-compiler-fatal-error-rw1016).  
  
  
RC terminating after preprocessor errors  
  
 The Resource Compiler halted due to other errors. Fix the other errors and recompile.
