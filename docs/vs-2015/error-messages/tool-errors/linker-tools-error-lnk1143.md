# Linker Tools Error LNK1143

The latest version of this topic can be found at [Linker Tools Error LNK1143](https://docs.microsoft.com/cpp/error-messages/tool-errors/linker-tools-error-lnk1143).  
  
  
invalid or corrupt file: no symbol for COMDAT section number
