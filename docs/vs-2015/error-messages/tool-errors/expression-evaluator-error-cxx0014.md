# Expression Evaluator Error CXX0014
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Expression Evaluator Error CXX0014](https://docs.microsoft.com/cpp/error-messages/tool-errors/expression-evaluator-error-cxx0014).  
  
  
missing operand  
  
 An operator was specified without a required operand. Check the syntax of the expression.  
  
 This error is identical to CAN0014.
