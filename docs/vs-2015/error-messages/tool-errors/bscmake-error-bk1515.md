# BSCMAKE Error BK1515
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [BSCMAKE Error BK1515](https://docs.microsoft.com/cpp/error-messages/tool-errors/bscmake-error-bk1515).  
  
  
bscfile: incompatible version: cannot incrementally update
