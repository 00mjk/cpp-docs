# Resource Compiler Error RC2127
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Resource Compiler Error RC2127](https://docs.microsoft.com/cpp/error-messages/tool-errors/resource-compiler-error-rc2127).  
  
  
version WORDs separated by commas expected  
  
 Version numbers in a version resource should be of type **WORD**, and separated by commas.
