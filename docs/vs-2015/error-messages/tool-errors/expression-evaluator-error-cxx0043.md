# Expression Evaluator Error CXX0043
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Expression Evaluator Error CXX0043](https://docs.microsoft.com/cpp/error-messages/tool-errors/expression-evaluator-error-cxx0043).  
  
  
this pointer used outside member function  
  
 The **this** pointer can only be used for nonstatic member functions.  
  
 This error is identical to CAN0043.
