# BSCMAKE Error BK1508
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [BSCMAKE Error BK1508](https://docs.microsoft.com/cpp/error-messages/tool-errors/bscmake-error-bk1508).  
  
  
cannot delete temporary file 'filename' [: reason]  
  
 BSCMAKE cannot delete one of its temporary files.  
  
### To fix by checking the following possible causes  
  
1.  Another process has the file open.  
  
2.  A hardware error.
