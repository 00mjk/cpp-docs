# Resource Compiler Error RC2122
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Resource Compiler Error RC2122](https://docs.microsoft.com/cpp/error-messages/tool-errors/resource-compiler-error-rc2122).  
  
  
unknown menu subtype  
  
 The *item-definition* field of the **MENU** statement can contain only **MENUITEM** and **POPUP** statements.
