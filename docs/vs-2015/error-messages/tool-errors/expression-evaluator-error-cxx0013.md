# Expression Evaluator Error CXX0013
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Expression Evaluator Error CXX0013](https://docs.microsoft.com/cpp/error-messages/tool-errors/expression-evaluator-error-cxx0013).  
  
  
missing operator  
  
 An operator was expected in the expression but was not found. Check the syntax of the expression.  
  
 This error is identical to CAN0013.
