# Resource Compiler Error RC2163
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Resource Compiler Error RC2163](https://docs.microsoft.com/cpp/error-messages/tool-errors/resource-compiler-error-rc2163).  
  
  
accelerator type required [ASCII or VIRTKEY]  
  
 The `type` field in the **ACCELERATORS** statement must contain either the ASCII or VIRTKEY value.
