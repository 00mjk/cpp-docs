# Linker Tools Warning LNK4216
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Linker Tools Warning LNK4216](https://docs.microsoft.com/cpp/error-messages/tool-errors/linker-tools-warning-lnk4216).  
  
  
Exported entry point entry  
  
 An entry point was exported from a DLL. The entry point of a DLL does not need to be exported.
