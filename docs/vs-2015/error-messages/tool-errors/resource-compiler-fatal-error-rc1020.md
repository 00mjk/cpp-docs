# Resource Compiler Fatal Error RC1020
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Resource Compiler Fatal Error RC1020](https://docs.microsoft.com/cpp/error-messages/tool-errors/resource-compiler-fatal-error-rc1020).  
  
  
unexpected '#endif'  
  
 An `#endif` directive appeared without a matching `#if`, **#ifdef**, or **#ifndef** directive.  
  
 Make sure that there is a matching `#endif` for each `#if`, **#ifdef**, and **#ifndef** statement.
