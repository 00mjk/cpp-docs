# Compiler Error C2526
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2526](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2526).  
  
  
identifier1' : C linkage function cannot return C++ class 'identifier2'  
  
 A function defined with C linkage cannot return a user-defined type.
