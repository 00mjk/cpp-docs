# Compiler Error C3506

The latest version of this topic can be found at [Compiler Error C3506](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c3506).  
  
  
there is no typelib registered for LIBID 'id'  
  
 A typelib was not registered correctly. Use regsvr32.exe to register a type library.
