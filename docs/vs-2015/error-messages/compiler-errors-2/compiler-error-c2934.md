# Compiler Error C2934
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2934](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2934).  
  
  
class' : type-class-id redefined as a nested 'item' of 'identifier'  
  
 You cannot use a generic or template class as a nested item.
