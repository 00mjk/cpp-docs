# Compiler Error C3820
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C3820](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c3820).  
  
  
name': initializers must be managed  
  
 The *name* initializer for a managed type must also be managed.  
  
### To correct this error  
  
1.  Specify a managed initializer method.
