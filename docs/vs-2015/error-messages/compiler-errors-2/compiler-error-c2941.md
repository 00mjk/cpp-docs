# Compiler Error C2941
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2941](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2941).  
  
  
class' : type-class-id redefined as a local 'item'  
  
 You cannot use a generic or template class as an item.
