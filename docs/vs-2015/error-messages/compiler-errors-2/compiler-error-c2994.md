# Compiler Error C2994
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2994](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2994).  
  
  
unnamed class in template parameter list  
  
 You cannot use the `class` keyword as a template argument without specifying a class name.
