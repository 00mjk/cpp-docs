# Compiler Error C2551
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2551](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2551).  
  
  
void *' type needs explicit cast  
  
 A `void` pointer is assigned to a nonvoid pointer by implicit conversion. You must use an explicit cast.
