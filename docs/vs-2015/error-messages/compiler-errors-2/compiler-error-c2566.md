# Compiler Error C2566

The latest version of this topic can be found at [Compiler Error C2566](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2566).  
  
  
overloaded function in conditional expression  
  
 An overloaded function in a conditional expression cannot be evaluated.
