# Compiler Error C2817
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2817](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2817).  
  
  
return type for 'operator delete' must be 'void'  
  
 An overloaded [operator delete](../Topic/operator%20delete%20\(%3Cnew%3E\).md) function cannot return a value.
