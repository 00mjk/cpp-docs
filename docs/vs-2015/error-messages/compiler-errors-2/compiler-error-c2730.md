# Compiler Error C2730
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2730](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2730).  
  
  
class' : cannot be a base class of itself  
  
 Recursive base classes are invalid. Specify another class as the base class.
