# Compiler Error C2544
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2544](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2544).  
  
  
expected ')' for operator '()'  
  
 The function call operator is missing a left parenthesis.  
  
 This error can be caused by macro expansion.
