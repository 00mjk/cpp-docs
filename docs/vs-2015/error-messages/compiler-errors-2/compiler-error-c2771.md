# Compiler Error C2771
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2771](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2771).  
  
  
import only permitted at global or namespace scope  
  
 The `#import` directive is not allowed in, for example, a function or structure.
