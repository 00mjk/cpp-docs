# Compiler Error C2543
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2543](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2543).  
  
  
expected ']' for operator '[]'  
  
 The subscripting operator is missing a left bracket.  
  
 This error can be caused by macro expansion.
