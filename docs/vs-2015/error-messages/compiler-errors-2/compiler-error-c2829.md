# Compiler Error C2829
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2829](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2829).  
  
  
operator operator' cannot have a variable parameter list  
  
 Only two operators can take variable parameter lists:  
  
-   Function call `()`  
  
-   `new`
