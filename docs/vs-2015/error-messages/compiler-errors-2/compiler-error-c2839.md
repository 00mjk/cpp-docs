# Compiler Error C2839
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [Compiler Error C2839](https://docs.microsoft.com/cpp/error-messages/compiler-errors-2/compiler-error-c2839).  
  
  
invalid return type 'type' for overloaded 'operator ->'  
  
 The `->` operator must return a class, struct, or union, or a reference to one.
