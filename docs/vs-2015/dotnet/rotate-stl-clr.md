# rotate (STL/CLR)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [rotate (STL/CLR)](https://docs.microsoft.com/cpp/dotnet/rotate-stl-clr).  
  
  
Exchanges the elements in two adjacent ranges.  
  
## Syntax  
  
```  
template<class _FwdIt> inline  
    void rotate(_FwdIt _First, _FwdIt _Mid, _FwdIt _Last);  
```  
  
## Remarks  
 This function behaves the same as the STL function `rotate`. For more information, see [rotate](../Topic/rotate.md).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
