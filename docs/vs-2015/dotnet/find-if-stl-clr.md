# find_if (STL/CLR)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [find_if (STL/CLR)](https://docs.microsoft.com/cpp/dotnet/find-if-stl-clr).  
  
  
Locates the position of the first occurrence of an element in a range that satisfies a specified condition.  
  
## Syntax  
  
```  
template<class _InIt, class _Pr> inline  
    _InIt find_if(_InIt _First, _InIt _Last, _Pr _Pred);  
```  
  
## Remarks  
 This function behaves the same as the STL function `find_if`. For more information, see [find_if](../Topic/find_if.md).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
