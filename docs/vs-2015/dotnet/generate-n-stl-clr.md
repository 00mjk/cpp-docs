# generate_n (STL/CLR)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [generate_n (STL/CLR)](https://docs.microsoft.com/cpp/dotnet/generate-n-stl-clr).  
  
  
Assigns the values generated by a function object to a specified number of element is a range and returns to the position one past the last assigned value.  
  
## Syntax  
  
```  
template<class _OutIt, class _Diff, class _Fn0> inline  
    void generate_n(_OutIt _Dest, _Diff _Count, _Fn0 _Func);  
```  
  
## Remarks  
 This function behaves the same as the STL function `generate_n`. For more information, see [generate_n](../Topic/generate_n.md).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
