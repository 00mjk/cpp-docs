# auto_handle Class
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [auto_handle Class](https://docs.microsoft.com/cpp/dotnet/auto-handle-class).  
  
  
Automatic resource management which can be used to embed a virtual handle into a managed type.  
  
## Syntax  
  
```  
template<typename _element_type>  
ref class auto_handle;  
```  
  
#### Parameters  
 `_element_type`  
 The managed type to be embedded.  
  
## Requirements  
 **Header file** \<msclr\auto_handle.h>  
  
 **Namespace** msclr
