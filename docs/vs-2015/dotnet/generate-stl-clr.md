# generate (STL/CLR)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [generate (STL/CLR)](https://docs.microsoft.com/cpp/dotnet/generate-stl-clr).  
  
  
Assigns the values generated by a function object to each element in a range.  
  
## Syntax  
  
```  
template<class _FwdIt, class _Fn0> inline  
    void generate(_FwdIt _First, _FwdIt _Last, _Fn0 _Func);  
```  
  
## Remarks  
 This function behaves the same as the STL function `generate`. For more information, see [generate](../Topic/generate.md).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
