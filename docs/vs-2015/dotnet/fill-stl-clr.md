# fill (STL/CLR)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [fill (STL/CLR)](https://docs.microsoft.com/cpp/dotnet/fill-stl-clr).  
  
  
Assigns the same new value to every element in a specified range.  
  
## Syntax  
  
```  
template<class _FwdIt, class _Ty> inline  
    void fill(_FwdIt _First, _FwdIt _Last, const _Ty% _Val);  
```  
  
## Remarks  
 This function behaves the same as the STL function `fill`. For more information, see [fill](http://msdn.microsoft.com/library/b1f09e99-a9d2-4e4c-97f8-aaccc20b4420).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
