# find (STL/CLR)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [find (STL/CLR)](https://docs.microsoft.com/cpp/dotnet/find-stl-clr).  
  
  
Locates the position of the first occurrence of an element in a range that has a specified value.  
  
## Syntax  
  
```  
template<class _InIt, class _Ty> inline  
    _InIt find(_InIt _First, _InIt _Last, const _Ty% _Val);  
```  
  
## Remarks  
 This function behaves the same as the STL function `find`. For more information, see [find](../Topic/find%20\(STL\).md).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
