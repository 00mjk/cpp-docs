# auto_gcroot Class
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [auto_gcroot Class](https://docs.microsoft.com/cpp/dotnet/auto-gcroot-class).  
  
  
Automatic resource management (like [auto_ptr Class](../standard-library/auto-ptr-class.md)) which can be used to embed a virtual handle into a native type.  
  
## Syntax  
  
```  
template<typename _element_type>  
class auto_gcroot;  
```  
  
#### Parameters  
 `_element_type`  
 The managed type to be embedded.  
  
## Requirements  
 **Header file** \<msclr\auto_gcroot.h>  
  
 **Namespace** msclr
