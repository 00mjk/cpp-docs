# Summary of String Literals
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Summary of String Literals](https://docs.microsoft.com/cpp/c-language/summary-of-string-literals).  
  
string-literal*:  
 **'** *s-char-sequence* opt**'**  
  
 **L'** *s-char-sequence* opt**'**  
  
 *s-char-sequence*:  
 *s-char*  
  
 *s-char-sequence s-char*  
  
 *s-char*:  
 any member of the source character set except the double-quotation mark ("), backslash (\\), or newline character *escape-sequence*
