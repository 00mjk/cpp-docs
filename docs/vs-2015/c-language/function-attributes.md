# Function Attributes
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Function Attributes](https://docs.microsoft.com/cpp/c-language/function-attributes).  
  
Microsoft Specific**  
  
 The optional *attribute-seq* nonterminal allows you to select a calling convention on a per-function basis. You can also specify functions as `__fastcall` or `__inline`.  
  
 **END Microsoft Specific**
