# ArgTraitsHelper::args Constant
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [ArgTraitsHelper::args Constant](https://docs.microsoft.com/cpp/windows/argtraitshelper-args-constant).  
  
  
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
static const int args = Traits::args;  
```  
  
## Remarks  
 Helps [ArgTraitsHelper::args](../windows/argtraitshelper-args-constant.md) keep count of the number of parameters on the Invoke method of a delegate interface.  
  
## Requirements  
 **Header:** event.h  
  
 **Namespace:** Microsoft::WRL::Details
