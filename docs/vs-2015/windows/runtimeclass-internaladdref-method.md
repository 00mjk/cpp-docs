# RuntimeClass::InternalAddRef Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [RuntimeClass::InternalAddRef Method](https://docs.microsoft.com/cpp/windows/runtimeclass-internaladdref-method).  
  
  
Increments the reference count to the current RuntimeClass object.  
  
## Syntax  
  
```  
ULONG InternalAddRef();  
```  
  
## Return Value  
 The resulting reference count.  
  
## Requirements  
 **Header:** implements.h  
  
 **Namespace:** Microsoft::WRL
