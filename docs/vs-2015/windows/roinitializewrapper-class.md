# RoInitializeWrapper Class
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [RoInitializeWrapper Class](https://docs.microsoft.com/cpp/windows/roinitializewrapper-class).  
  
  
Initializes the [!INCLUDE[wrt](../includes/wrt-md.md)].  
  
## Syntax  
  
```  
class RoInitializeWrapper  
```  
  
## Remarks  
 RoInitializeWrapper is a convenience that initializes the [!INCLUDE[wrt](../includes/wrt-md.md)] and returns an HRESULT that indicates whether the operation was successful.
