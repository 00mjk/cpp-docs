# Module::objectCount_ Data Member
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Module::objectCount_ Data Member](https://docs.microsoft.com/cpp/windows/module-objectcount-data-member).  
  
  
Keeps track of how many classes have been created with the [Make](../windows/make-function.md) function.  
  
## Syntax  
  
```  
volatile long objectCount_;  
```
