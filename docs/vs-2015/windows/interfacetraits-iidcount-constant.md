# InterfaceTraits::IidCount Constant
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [InterfaceTraits::IidCount Constant](https://docs.microsoft.com/cpp/windows/interfacetraits-iidcount-constant).  
  
  
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
static const unsigned long IidCount = 1;  
```  
  
## Remarks  
 Holds the number of interface IDs associated with the current InterfaceTraits object.  
  
## Requirements  
 **Header:** implements.h  
  
 **Namespace:** Microsoft::WRL::Details
