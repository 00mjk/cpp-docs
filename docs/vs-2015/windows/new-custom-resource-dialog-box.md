# New Custom Resource Dialog Box
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [New Custom Resource Dialog Box](https://docs.microsoft.com/cpp/windows/new-custom-resource-dialog-box).  
  
  
Enables you to create a new custom resource.  
  
 **Resource Type**  
 Provides a text box for you to enter the name of a custom resource type. Visual C++ automatically capitalizes the name when you exit the **New Custom Resource** dialog box.  
  
## Remarks  
 Do not specify a resource name or [!INCLUDE[TLA2#tla_id](../includes/tla2sharptla-id-md.md)] that is reserved by [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]. The reserved names are DESIGNINFO, HWB, and TEXTINCLUDE, and the reserved [!INCLUDE[TLA2#tla_id](../includes/tla2sharptla-id-md.md)] is 255.
