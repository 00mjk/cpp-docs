# SyncLockT::~SyncLockT Destructor
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [SyncLockT::~SyncLockT Destructor](https://docs.microsoft.com/cpp/windows/synclockt-tilde-synclockt-destructor).  
  
  
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
~SyncLockT();  
```  
  
## Remarks  
 Deinitializes an instance of the SyncLockT class.  
  
 This destructor also unlocks the current SyncLockT instance.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers::Details
