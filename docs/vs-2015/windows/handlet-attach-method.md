# HandleT::Attach Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [HandleT::Attach Method](https://docs.microsoft.com/cpp/windows/handlet-attach-method).  
  
  
Associates the specified handle with the current HandleT object.  
  
## Syntax  
  
```  
void Attach(  
   typename HandleTraits::Type h  
);  
```  
  
#### Parameters  
 `h`  
 A handle.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
