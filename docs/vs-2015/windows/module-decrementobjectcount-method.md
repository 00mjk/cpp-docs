# Module::DecrementObjectCount Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Module::DecrementObjectCount Method](https://docs.microsoft.com/cpp/windows/module-decrementobjectcount-method).  
  
  
Decrements the number of objects tracked by the module.  
  
## Syntax  
  
```  
virtual long DecrementObjectCount();  
```  
  
## Return Value  
 The count before the decrement operation.  
  
## Requirements  
 **Header:** module.h  
  
 **Namespace:** Microsoft::WRL
