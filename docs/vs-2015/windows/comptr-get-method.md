# ComPtr::Get Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [ComPtr::Get Method](https://docs.microsoft.com/cpp/windows/comptr-get-method).  
  
  
Retrieves a pointer to the interface that is associated with this ComPtr.  
  
## Syntax  
  
```  
T* Get() const;  
```  
  
## Return Value  
 Pointer to the interface that is associated with this ComPtr.  
  
## Requirements  
 **Header:** client.h  
  
 **Namespace:** Microsoft::WRL
