# BoolStruct::Member Data Member
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [BoolStruct::Member Data Member](https://docs.microsoft.com/cpp/windows/boolstruct-member-data-member).  
  
  
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
int Member;  
```  
  
## Remarks  
 Specifies that a [ComPtr](../windows/comptr-class.md) is, or is not, managing the object lifetime of an interface.
