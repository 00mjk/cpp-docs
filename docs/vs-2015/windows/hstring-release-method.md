# HString::Release Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [HString::Release Method](https://docs.microsoft.com/cpp/windows/hstring-release-method).  
  
  
Deletes the underlying string value and intializes the current HString object to an empty value.  
  
## Syntax  
  
```  
void Release() throw()  
```  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
