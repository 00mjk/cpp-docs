# SafeInt Library
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [SafeInt Library](https://docs.microsoft.com/cpp/windows/safeint-library).  
  
The SafeInt library helps prevent integer overflows that might result when the application performs mathematical operations.
