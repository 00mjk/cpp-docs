# Displaying or Hiding the Colors Window (Image Editor for Icons)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Displaying or Hiding the Colors Window (Image Editor for Icons)](https://docs.microsoft.com/cpp/windows/displaying-or-hiding-the-colors-window-image-editor-for-icons).  
  
  
To display Colors window  
  
1.  Right-click in an **Image Editor** pane and choose **Show Colors Window** from the shortcut menu.  
  
     \- or -  
  
2.  Click **Show Colors Window** on the [Image menu](../mfc/image-menu-image-editor-for-icons.md).  
  
### To hide the Colors window  
  
1.  Unpin the window. This will allow the window to auto hide when it's not in use. For more information, see [Arranging Windows](../misc/how-to-arrange-and-dock-windows.md).  
  
 \- or -  
  
-   Click the **Close** button.  
  
 Requirements  
  
 None
