# HString::Attach Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [HString::Attach Method](https://docs.microsoft.com/cpp/windows/hstring-attach-method).  
  
  
Associates the specified HString object with the current HString object.  
  
## Syntax  
  
```  
  
void Attach(  
       HSTRING hstr  
       ) throw()  
```  
  
#### Parameters  
 `hstr`  
 An existing HString object.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
