# SRWLockExclusiveTraits::GetInvalidValue Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [SRWLockExclusiveTraits::GetInvalidValue Method](https://docs.microsoft.com/cpp/windows/srwlockexclusivetraits-getinvalidvalue-method).  
  
  
Retrieves an SRWLockExclusiveTraits object that is always invalid.  
  
## Syntax  
  
```  
inline static Type GetInvalidValue();  
```  
  
## Return Value  
 An empty SRWLockExclusiveTraits object.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers::HandleTraits
