# HandleT::IsValid Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [HandleT::IsValid Method](https://docs.microsoft.com/cpp/windows/handlet-isvalid-method).  
  
  
Indicates whether the current HandleT object represents a handle.  
  
## Syntax  
  
```  
bool IsValid() const;  
```  
  
## Return Value  
 `true` if the HandleT represents a handle; otherwise, `false`.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
