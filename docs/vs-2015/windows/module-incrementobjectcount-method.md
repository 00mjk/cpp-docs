# Module::IncrementObjectCount Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Module::IncrementObjectCount Method](https://docs.microsoft.com/cpp/windows/module-incrementobjectcount-method).  
  
  
Increments the number of objects tracked by the module.  
  
## Syntax  
  
```  
virtual long IncrementObjectCount();  
```  
  
## Return Value  
 The count before the increment operation.  
  
## Requirements  
 **Header:** module.h  
  
 **Namespace:** Microsoft::WRL
