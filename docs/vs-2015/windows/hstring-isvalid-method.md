# HString::IsValid Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [HString::IsValid Method](https://docs.microsoft.com/cpp/windows/hstring-isvalid-method).  
  
  
Indicates whether the current HString object is empty or not.  
  
## Syntax  
  
```  
bool IsValid() const throw()  
```  
  
#### Parameters  
 `true` if the current HString object is not empty; otherwise, `false`.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
