# CriticalSection::IsValid Method
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [CriticalSection::IsValid Method](https://docs.microsoft.com/cpp/windows/criticalsection-isvalid-method).  
  
  
Indicates whether the current critical section is valid.  
  
## Syntax  
  
```  
bool IsValid() const;  
```  
  
## Return Value  
 By default, always returns `true`.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
