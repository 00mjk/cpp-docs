# CRowset::ReleaseRows
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CRowset::ReleaseRows](https://docs.microsoft.com/cpp/data/oledb/crowset-releaserows).  
  
  
Calls [IRowset::ReleaseRows](https://msdn.microsoft.com/library/ms719771.aspx) to release the current row handle.  
  
## Syntax  
  
```  
  
HRESULT ReleaseRows( ) throw( );  
  
```  
  
## Return Value  
 A standard `HRESULT`.  
  
## Requirements  
 **Header:** atldbcli.h
