# CDynamicAccessor::Close
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CDynamicAccessor::Close](https://docs.microsoft.com/cpp/data/oledb/cdynamicaccessor-close).  
  
  
Unbinds all the columns, releases the allocated memory, and releases the [IAccessor](https://msdn.microsoft.com/library/ms719672.aspx) interface pointer in the class.  
  
## Syntax  
  
```  
  
void Close( ) throw( );  
  
```  
  
## Requirements  
 **Header:** atldbcli.h
