# CAccessorRowset::Bind
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CAccessorRowset::Bind](https://docs.microsoft.com/cpp/data/oledb/caccessorrowset-bind).  
  
  
Creates the bindings if you specified **bBind** as false in [CCommand::Open](../../data/oledb/ccommand-open.md).  
  
## Syntax  
  
```  
  
HRESULT Bind( );  
  
```  
  
## Return Value  
 A standard `HRESULT`.  
  
## Requirements  
 **Header:** atldbcli.h
