# CDynamicParameterAccessor::GetParamCount
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CDynamicParameterAccessor::GetParamCount](https://docs.microsoft.com/cpp/data/oledb/cdynamicparameteraccessor-getparamcount).  
  
  
Retrieves the number of parameters stored in the buffer.  
  
## Syntax  
  
```  
  
DB_UPARAMS GetParamCount( ) const throw( );  
  
```  
  
## Return Value  
 The number of parameters.  
  
## Requirements  
 **Header:** atldbcli.h
