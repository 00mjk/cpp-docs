# CRowset::Close
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CRowset::Close](https://docs.microsoft.com/cpp/data/oledb/crowset-close).  
  
  
Releases rows and the current [IRowset](https://msdn.microsoft.com/library/ms720986.aspx) interface.  
  
## Syntax  
  
```  
  
void Close( ) throw( );  
  
```  
  
## Remarks  
 This method releases all rows currently in the rowset.  
  
## Requirements  
 **Header:** atldbcli.h
