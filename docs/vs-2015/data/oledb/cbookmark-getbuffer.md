# CBookmark::GetBuffer
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CBookmark::GetBuffer](https://docs.microsoft.com/cpp/data/oledb/cbookmark-getbuffer).  
  
  
Retrieves the pointer to the bookmark buffer.  
  
## Syntax  
  
```  
  
virtual BYTE* GetBuffer( ) const throw( );  
  
```  
  
## Return Value  
 A pointer to the bookmark buffer.  
  
## Requirements  
 **Header:** atldbcli.h
