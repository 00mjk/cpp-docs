# CDynamicAccessor::GetColumnCount
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CDynamicAccessor::GetColumnCount](https://docs.microsoft.com/cpp/data/oledb/cdynamicaccessor-getcolumncount).  
  
  
Retrieves the number of columns.  
  
## Syntax  
  
```  
  
DBORDINAL GetColumnCount( ) const throw( );  
  
```  
  
## Return Value  
 The number of columns retrieved.

 **Header:** atldbcli.h
