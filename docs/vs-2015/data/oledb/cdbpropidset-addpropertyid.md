# CDBPropIDSet::AddPropertyID
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CDBPropIDSet::AddPropertyID](https://docs.microsoft.com/cpp/data/oledb/cdbpropidset-addpropertyid).  
  
  
Adds a property ID to the property ID set.  
  
## Syntax  
  
```  
  
      bool AddPropertyID(   
   DBPROPID propid    
) throw( );  
```  
  
#### Parameters  
 `propid`  
 [in] The property ID to be added to the property ID set.  
  
## Requirements  
 **Header:** atldbcli.h
