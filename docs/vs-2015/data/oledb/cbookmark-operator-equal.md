The latest version of this topic can be found at [CBookmark::operator =](https://docs.microsoft.com/cpp/data/oledb/cbookmark-operator-equal).

## Syntax  
  
```  
  
      CBookmark& operator =(   
   const CBookmark& bookmark    
) throw( );  
```
