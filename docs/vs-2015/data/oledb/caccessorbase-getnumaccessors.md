# CAccessorBase::GetNumAccessors
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CAccessorBase::GetNumAccessors](https://docs.microsoft.com/cpp/data/oledb/caccessorbase-getnumaccessors).  
  
  
Retrieves the number of accessors created by the class.  
  
## Syntax  
  
```  
  
ULONG GetNumAccessors( ) const;  
  
```  
  
## Return Value  
 The number of accessors created by the class.  
  
## Requirements  
 **Header:** atldbcli.h
