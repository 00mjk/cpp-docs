# CAccessorBase::Close
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [CAccessorBase::Close](https://docs.microsoft.com/cpp/data/oledb/caccessorbase-close).  
  
  
Closes the accessors.  
  
## Syntax  
  
```  
  
void Close( );  
  
```  
  
## Remarks  
 You must call [ReleaseAccessors](../../data/oledb/caccessorbase-releaseaccessors.md) first.
