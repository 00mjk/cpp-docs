# Copying Buttons from a Toolbar
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Copying Buttons from a Toolbar](https://docs.microsoft.com/cpp/windows/copying-buttons-from-a-toolbar).  
  
  
To copy buttons from a toolbar  
  
1.  Hold down the **CTRL** key.  
  
2.  In the **Toolbar View** pane, drag the button to either its new location on the toolbar or to a location on another toolbar.
