# Environmental Constants
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Environmental Constants](https://docs.microsoft.com/cpp/c-runtime-library/environmental-constants).  
  
Syntax  
  
```

```  
  
## Remarks  
 This constant defines the environmental length for strings.  
  
|Constant|Meaning|  
|--------------|-------------|  
|`_MAX_ENV`|Maximum string size of an environmental string.|
