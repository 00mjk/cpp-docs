# RAND_MAX
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [RAND_MAX](https://docs.microsoft.com/cpp/c-runtime-library/rand-max).  
  
Syntax  
  
```  
  
#include <stdlib.h>  
  
```  
  
## Remarks  
 The constant `RAND_MAX` is the maximum value that can be returned by the `rand` function. `RAND_MAX` is defined as the value 0x7fff.
