# TMP_MAX, L_tmpnam
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [TMP_MAX, L_tmpnam](https://docs.microsoft.com/cpp/c-runtime-library/tmp-max-l-tmpnam).  
  
Syntax  
  
```  
#include <stdio.h>  
```  
  
## Remarks  
 `TMP_MAX` is the maximum number of unique filenames that the `tmpnam` function can generate. `L_tmpnam` is the length of temporary filenames generated by `tmpnam`.
