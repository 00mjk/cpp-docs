# __int2c
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [__int2c](https://docs.microsoft.com/cpp/intrinsics/int2c).  
  
  
Microsoft Specific**  
  
 Generates the `int 2c` instruction, which triggers the `2c` interrupt.  
  
## Syntax  
  
```  
void __int2c(void);  
```  
  
## Requirements  
  
|Intrinsic|Architecture|  
|---------------|------------------|  
|`__int2c`|x86, [!INCLUDE[vcprx64](../includes/vcprx64-md.md)]|
