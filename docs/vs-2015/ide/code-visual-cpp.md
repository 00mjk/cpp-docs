# &lt;code&gt; (Visual C++)
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [&lt;code&gt; (Visual C++)](https://docs.microsoft.com/cpp/ide/code-visual-cpp).  
  
  
The \<code> tag gives you a way to indicate one or more lines as code.  
  
## Syntax  
  
```  
<code>content</code>  
```  
  
#### Parameters  
 `content`  
 The text you want marked as code.  
  
## Remarks  
 Use [\<c>](../ide/c-visual-cpp.md) to indicate a portion of text should be marked as code.  
  
 Compile with [/doc](../build/reference/doc-process-documentation-comments-c-cpp.md) to process documentation comments to a file.
