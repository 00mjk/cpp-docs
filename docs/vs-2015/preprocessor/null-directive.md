# Null Directive
[!INCLUDE[blank_token](../includes/blank-token.md)]

The latest version of this topic can be found at [Null Directive](https://docs.microsoft.com/cpp/preprocessor/null-directive).  
  
The null preprocessor directive is a single number sign (**#**) alone on a line. It has no effect.  
  
## Syntax  
  
```  
  
#  
  
```
