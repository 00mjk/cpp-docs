# operator THIS
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator THIS](https://docs.microsoft.com/cpp/assembler/masm/operator-this).  
  
  
Returns an operand of specified `type` whose offset and segment values are equal to the current location counter value.  
  
## Syntax  
  
```  
  
THIS type  
```
