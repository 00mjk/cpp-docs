# @DataSize
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [@DataSize](https://docs.microsoft.com/cpp/assembler/masm/at-datasize).  
  
  
0 for **TINY**, **SMALL**, **MEDIUM**, and **FLAT** models, 1 for **COMPACT** and **LARGE** models, and 2 for **HUGE** model (numeric equate).  
  
## Syntax  
  
```  
  
@DataSize  
  
```
