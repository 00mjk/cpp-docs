# .ERRNDEF
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.ERRNDEF](https://docs.microsoft.com/cpp/assembler/masm/dot-errndef).  
  
  
Generates an error if *name* has not been defined.  
  
## Syntax  
  
```  
  
.ERRNDEF name [[, message]]  
```
