# operator ! (MASM)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator ! (MASM)](https://docs.microsoft.com/cpp/assembler/masm/operator-logical-not-masm).  
  
  
Treats `character` as a literal character rather than as an operator or symbol.  
  
## Syntax  
  
```  
  
!  
character  
  
```
