# ML Warning A4004
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [ML Warning A4004](https://docs.microsoft.com/cpp/assembler/masm/ml-warning-a4004).  
  
  
cannot ASSUME CS**  
  
 An attempt was made to assume a value for the CS register. CS is always set to the current segment or group.
