# operator SIZE
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator SIZE](https://docs.microsoft.com/cpp/assembler/masm/operator-size).  
  
  
Returns the number of bytes in *variable* allocated by the first initializer.  
  
## Syntax  
  
```  
  
SIZE variable  
```
