# operator SIZEOF
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator SIZEOF](https://docs.microsoft.com/cpp/assembler/masm/operator-sizeof).  
  
  
Returns the number of bytes in *variable* or `type`.  
  
## Syntax  
  
```  
  
SIZEOF {variable | type}  
```
