# operator MASK
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator MASK](https://docs.microsoft.com/cpp/assembler/masm/operator-mask).  
  
  
Returns a bit mask in which the bits in *recordfieldname* or *record* are set and all other bits are cleared.  
  
## Syntax  
  
```  
  
MASK {recordfieldname | record}  
```
