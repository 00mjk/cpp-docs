# .NOLIST
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.NOLIST](https://docs.microsoft.com/cpp/assembler/masm/dot-nolist).  
  
  
Suppresses program listing.  
  
## Syntax  
  
```  
  
.NOLIST  
  
```  
  
## Remarks  
 Same as [.XLIST](../../assembler/masm/dot-xlist.md).
