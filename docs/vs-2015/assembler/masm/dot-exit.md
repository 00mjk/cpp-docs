# .EXIT
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.EXIT](https://docs.microsoft.com/cpp/assembler/masm/dot-exit).  
  
  
Generates termination code.  
  
## Syntax  
  
```  
  
.EXIT [[expression]]  
```  
  
## Remarks  
 Returns optional *expression* to shell.
