# .ERRNZ
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.ERRNZ](https://docs.microsoft.com/cpp/assembler/masm/dot-errnz).  
  
  
Generates an error if *expression* is true (nonzero).  
  
## Syntax  
  
```  
  
.ERRNZ expression [[, message]]  
```
