# ALIGN (MASM)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [ALIGN (MASM)](https://docs.microsoft.com/cpp/assembler/masm/align-masm).  
  
  
Aligns the next variable or instruction on a byte that is a multiple of `number`.  
  
## Syntax  
  
```  
  
ALIGN [[  
number  
]]  
  
```
