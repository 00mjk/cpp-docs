# ENDM
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [ENDM](https://docs.microsoft.com/cpp/assembler/masm/endm).  
  
  
Terminates a macro or repeat block.  
  
## Syntax  
  
```  
  
ENDM  
  
```
