# ECHO
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [ECHO](https://docs.microsoft.com/cpp/assembler/masm/echo).  
  
  
Displays *message* to the standard output device (by default, the screen).  
  
## Syntax  
  
```  
  
ECHO   
message  
  
```  
  
## Remarks  
 Same as [%OUT](../../assembler/masm/percent-out.md).
