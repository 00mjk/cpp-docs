# operator LOWWORD
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator LOWWORD](https://docs.microsoft.com/cpp/assembler/masm/operator-lowword).  
  
  
Returns the low 16 bits of *expression*. MASM expressions are 64-bit values.  
  
## Syntax  
  
```  
  
LOWWORD expression  
```
