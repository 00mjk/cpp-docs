# operator HIGH
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator HIGH](https://docs.microsoft.com/cpp/assembler/masm/operator-high).  
  
  
Returns the high 8 bits of the low 16 bits of *expression*. MASM expressions are 64-bit values.  
  
## Syntax  
  
```  
  
HIGH expression  
```
