# operator %
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator %](https://docs.microsoft.com/cpp/assembler/masm/operator-percent).  
  
  
Treats the value of *expression* in a macro argument as text.  
  
## Syntax  
  
```  
  
%  
expression  
  
```
