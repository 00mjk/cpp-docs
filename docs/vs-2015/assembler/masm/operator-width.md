# operator WIDTH
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator WIDTH](https://docs.microsoft.com/cpp/assembler/masm/operator-width).  
  
  
Returns the width in bits of the current *recordfieldname* or *record*.  
  
## Syntax  
  
```  
  
WIDTH {recordfieldname | record}  
```
