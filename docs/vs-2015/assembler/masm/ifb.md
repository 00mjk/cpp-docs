# IFB
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [IFB](https://docs.microsoft.com/cpp/assembler/masm/ifb).  
  
  
Grants assembly if *textitem* is blank.  
  
## Syntax  
  
```  
  
IFB   
textitem  
  
```
