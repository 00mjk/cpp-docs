# operator LROFFSET
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator LROFFSET](https://docs.microsoft.com/cpp/assembler/masm/operator-lroffset).  
  
  
Returns the offset of *expression*. Same as **OFFSET**, but it generates a loader resolved offset, which allows Windows to relocate code segments.  
  
## Syntax  
  
```  
  
LROFFSET expression  
```
