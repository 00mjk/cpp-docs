# BYTE (MASM)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [BYTE (MASM)](https://docs.microsoft.com/cpp/assembler/masm/byte-masm).  
  
  
Allocates and optionally initializes a byte of storage for each `initializer`.  
  
## Syntax  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## Remarks  
 Can also be used as a type specifier anywhere a type is legal.
