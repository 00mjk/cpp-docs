# .LISTIF
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.LISTIF](https://docs.microsoft.com/cpp/assembler/masm/dot-listif).  
  
  
Starts listing of statements in false conditional blocks.  
  
## Syntax  
  
```  
  
.LISTIF  
  
```  
  
## Remarks  
 Same as [.LFCOND](../../assembler/masm/dot-lfcond.md).
