# .FARDATA?
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.FARDATA?](https://docs.microsoft.com/cpp/assembler/masm/dot-fardata-q).  
  
  
When used with [.MODEL](../../assembler/masm/dot-model.md), starts a far data segment for uninitialized data (segment name FAR_BSS or *name*).  
  
## Syntax  
  
```  
  
.FARDATA? [[name]]  
```
