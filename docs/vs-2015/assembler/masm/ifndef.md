# IFNDEF
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [IFNDEF](https://docs.microsoft.com/cpp/assembler/masm/ifndef).  
  
  
Grants assembly if *name* has not been defined.  
  
## Syntax  
  
```  
  
IFNDEF name  
```
