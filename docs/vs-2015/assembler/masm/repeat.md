# REPEAT
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [REPEAT](https://docs.microsoft.com/cpp/assembler/masm/repeat).  
  
  
Marks a block that is to be repeated *expression* times. Same as [REPT](../../assembler/masm/rept.md).  
  
## Syntax  
  
```  
  
   REPEAT expression  
statements  
ENDM  
```
