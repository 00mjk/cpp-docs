# .ERRDEF
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.ERRDEF](https://docs.microsoft.com/cpp/assembler/masm/dot-errdef).  
  
  
Generates an error if *name* is a previously defined label, variable, or symbol.  
  
## Syntax  
  
```  
  
.ERRDEF name [[, message]]  
```
