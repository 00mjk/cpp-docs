# EVEN
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [EVEN](https://docs.microsoft.com/cpp/assembler/masm/even).  
  
  
Aligns the next variable or instruction on an even byte.  
  
## Syntax  
  
```  
  
EVEN  
  
```
