[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator OFFSET](https://docs.microsoft.com/cpp/assembler/masm/operator-offset).

## Syntax  
  
```  
  
OFFSET expression
```  
