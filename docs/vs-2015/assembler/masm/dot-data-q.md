# .DATA?
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.DATA?](https://docs.microsoft.com/cpp/assembler/masm/dot-data-q).  
  
  
When used with [.MODEL](../../assembler/masm/dot-model.md), starts a near data segment for uninitialized data (segment name _BSS).  
  
## Syntax  
  
```  
  
.DATA?  
  
```
