# IF2
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [IF2](https://docs.microsoft.com/cpp/assembler/masm/if2).  
  
  
IF** block is evaluated on every assembly pass if **OPTION:SETIF2** is **TRUE**.  
  
## Syntax  
  
```  
  
IF2   
expression  
  
```
