# OWORD
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [OWORD](https://docs.microsoft.com/cpp/assembler/masm/oword).  
  
  
Used as a type specifier when an 16-byte data type is required.  
  
## Syntax  
  
```  
  
OWORD  
  
```
