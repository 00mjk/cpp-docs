# PURGE
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [PURGE](https://docs.microsoft.com/cpp/assembler/masm/purge).  
  
  
Deletes the specified macros from memory.  
  
## Syntax  
  
```  
  
PURGE macroname [[, macroname]]...  
```
