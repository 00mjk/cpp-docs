# .NOLISTIF
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.NOLISTIF](https://docs.microsoft.com/cpp/assembler/masm/dot-nolistif).  
  
  
Suppresses listing of conditional blocks whose condition evaluates to false (0).  
  
## Syntax  
  
```  
  
.NOLISTIF  
  
```  
  
## Remarks  
 This is the default. Same as [.SFCOND](../../assembler/masm/dot-sfcond.md).
