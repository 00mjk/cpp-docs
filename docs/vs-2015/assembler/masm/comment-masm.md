# COMMENT (MASM)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [COMMENT (MASM)](https://docs.microsoft.com/cpp/assembler/masm/comment-masm).  
  
  
Treats all *text* between or on the same line as the delimiters as a comment.  
  
## Syntax  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
