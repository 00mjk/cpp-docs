# operator LENGTH
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator LENGTH](https://docs.microsoft.com/cpp/assembler/masm/operator-length).  
  
  
Returns the number of data items in *variable* created by the first initializer.  
  
## Syntax  
  
```  
  
LENGTH variable  
```
