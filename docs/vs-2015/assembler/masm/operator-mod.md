# operator MOD
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator MOD](https://docs.microsoft.com/cpp/assembler/masm/operator-mod).  
  
  
Returns the integer value of the remainder (modulo) when dividing *expression1* by *expression2*.  
  
## Syntax  
  
```  
  
expression1 MOD expression2  
```
