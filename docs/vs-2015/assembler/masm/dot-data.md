# .DATA
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.DATA](https://docs.microsoft.com/cpp/assembler/masm/dot-data).  
  
  
When used with [.MODEL](../../assembler/masm/dot-model.md), starts a near data segment for initialized data (segment name _DATA).  
  
## Syntax  
  
```  
.DATA  
```
