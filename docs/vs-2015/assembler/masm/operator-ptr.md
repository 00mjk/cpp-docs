# operator PTR
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator PTR](https://docs.microsoft.com/cpp/assembler/masm/operator-ptr).  
  
  
The first operator forces the *expression* to be treated as having the specified `type`. The second operator specifies a pointer to `type`.  
  
## Syntax  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
