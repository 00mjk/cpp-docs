# operator &amp;&amp; (MASM)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator &amp;&amp; (MASM)](https://docs.microsoft.com/cpp/assembler/masm/operator-logical-and-masm).  
  
  
Replaces `parameter` with its corresponding argument value.  
  
## Syntax  
  
```  
  
&  
parameter  
&  
  
```
