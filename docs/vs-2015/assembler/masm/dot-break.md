# .BREAK
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.BREAK](https://docs.microsoft.com/cpp/assembler/masm/dot-break).  
  
  
Generates code to terminate a [.WHILE](../../assembler/masm/dot-while.md) or [.REPEAT](../../assembler/masm/dot-repeat.md) block if *condition* is true.  
  
## Syntax  
  
```  
  
.BREAK [[.IF condition]]   
```
