# @CodeSize
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [@CodeSize](https://docs.microsoft.com/cpp/assembler/masm/at-codesize).  
  
  
0 for **TINY**, **SMALL**, **COMPACT**, and **FLAT** models, and 1 for **MEDIUM**, **LARGE**, and **HUGE** models (numeric equate).  
  
## Syntax  
  
```  
  
@CodeSize  
  
```
