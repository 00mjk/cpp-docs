# .NOLISTMACRO
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.NOLISTMACRO](https://docs.microsoft.com/cpp/assembler/masm/dot-nolistmacro).  
  
  
Suppresses listing of macro expansions.  
  
## Syntax  
  
```  
  
.NOLISTMACRO  
  
```  
  
## Remarks  
 Same as [.SALL](../../assembler/masm/dot-sall.md).
