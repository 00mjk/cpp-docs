# =
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [=](https://docs.microsoft.com/cpp/assembler/masm/equal).  
  
  
Assigns the numeric value of *expression* to *name*.  
  
## Syntax  
  
```  
  
name  
 =   
expression  
  
```  
  
## Remarks  
 The symbol can be redefined later.
