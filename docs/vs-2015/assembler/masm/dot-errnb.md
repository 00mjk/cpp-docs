# .ERRNB
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.ERRNB](https://docs.microsoft.com/cpp/assembler/masm/dot-errnb).  
  
  
Generates an error if *textitem* is not blank.  
  
## Syntax  
  
```  
  
.ERRNB <textitem> [[, message]]  
```
