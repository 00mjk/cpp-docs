# END (MASM)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [END (MASM)](https://docs.microsoft.com/cpp/assembler/masm/end-masm).  
  
  
Marks the end of a module and, optionally, sets the program entry point to *address*.  
  
## Syntax  
  
```  
  
END [[address]]  
```
