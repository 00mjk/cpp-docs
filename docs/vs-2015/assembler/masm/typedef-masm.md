# TYPEDEF (MASM)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [TYPEDEF (MASM)](https://docs.microsoft.com/cpp/assembler/masm/typedef-masm).  
  
  
Defines a new type called *name*, which is equivalent to `type`.  
  
## Syntax  
  
```  
  
name TYPEDEF type  
```
