# operator LOW32
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator LOW32](https://docs.microsoft.com/cpp/assembler/masm/operator-low32).  
  
  
Returns the low 32 bits of *expression*. MASM expressions are 64-bit values.  
  
## Syntax  
  
```  
  
LOW32 expression  
```
