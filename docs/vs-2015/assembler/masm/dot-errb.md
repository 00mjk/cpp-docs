# .ERRB
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.ERRB](https://docs.microsoft.com/cpp/assembler/masm/dot-errb).  
  
  
Generates an error if *textitem* is blank.  
  
## Syntax  
  
```  
  
.ERRB <textitem> [[, message]]  
```
