# operator HIGH32
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator HIGH32](https://docs.microsoft.com/cpp/assembler/masm/operator-high32).  
  
  
Returns the high 32 bits of *expression*. MASM expressions are 64-bit values.  
  
## Syntax  
  
```  
  
HIGH32 expression  
```
