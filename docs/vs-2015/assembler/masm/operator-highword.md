# operator HIGHWORD
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator HIGHWORD](https://docs.microsoft.com/cpp/assembler/masm/operator-highword).  
  
  
Returns the high 16 bits of the low 32 bits of *expression*. MASM expressions are 64-bit values.  
  
## Syntax  
  
```  
  
HIGHWORD expression  
```
