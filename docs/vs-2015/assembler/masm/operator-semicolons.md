# operator ;;
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator ;;](https://docs.microsoft.com/cpp/assembler/masm/operator-semicolons).  
  
  
Treats *text* as a comment in a macro that appears only in the macro definition. The listing does not show *text* where the macro is expanded.  
  
## Syntax  
  
```  
  
;;  
text  
  
```
