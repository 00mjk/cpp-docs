# operator DUP
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator DUP](https://docs.microsoft.com/cpp/assembler/masm/operator-dup).  
  
  
Specifies *count* number of declarations of *initialvalue*.  
  
## Syntax  
  
```  
  
count DUP (initialvalue [[, initialvalue]]...)  
```
