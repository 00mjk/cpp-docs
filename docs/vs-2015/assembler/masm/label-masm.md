# LABEL (MASM)
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [LABEL (MASM)](https://docs.microsoft.com/cpp/assembler/masm/label-masm).  
  
  
Creates a new label by assigning the current location-counter value and the given `type` to *name*.  
  
## Syntax  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
