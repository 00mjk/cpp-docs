# operator LOW
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [operator LOW](https://docs.microsoft.com/cpp/assembler/masm/operator-low).  
  
  
Returns the low 8 bits of *expression*. MASM expressions are 64-bit values.  
  
## Syntax  
  
```  
  
LOW expression  
```
