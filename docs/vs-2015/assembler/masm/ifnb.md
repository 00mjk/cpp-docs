# IFNB
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [IFNB](https://docs.microsoft.com/cpp/assembler/masm/ifnb).  
  
  
Grants assembly if *textitem* is not blank.  
  
## Syntax  
  
```  
  
IFNB textitem  
```
