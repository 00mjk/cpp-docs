# .FARDATA
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.FARDATA](https://docs.microsoft.com/cpp/assembler/masm/dot-fardata).  
  
  
When used with [.MODEL](../../assembler/masm/dot-model.md), starts a far data segment for initialized data (segment name FAR_DATA or *name*).  
  
## Syntax  
  
```  
  
.FARDATA [[name]]  
```
