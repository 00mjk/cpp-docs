# .LISTMACROALL
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.LISTMACROALL](https://docs.microsoft.com/cpp/assembler/masm/dot-listmacroall).  
  
  
Starts listing of all statements in macros.  
  
## Syntax  
  
```  
  
.LISTMACROALL  
  
```  
  
## Remarks  
 Same as [.LALL](../../assembler/masm/dot-lall.md).
