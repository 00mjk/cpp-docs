# __int2c
**Microsoft Specific**  
  
 Generates the `int 2c` instruction, which triggers the `2c` interrupt.  
  
## Syntax  
  
```  
void __int2c(void);  
```  
  
## Requirements  
  
|Intrinsic|Architecture|  
|---------------|------------------|  
|`__int2c`|x86, [!INCLUDE[vcprx64](../build/includes/vcprx64_md.md)]|  
  
 **Header file** \<intrin.h>
