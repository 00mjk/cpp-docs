# operator ZERO?
Status of zero flag. Used only within [.IF](../intrinsics/.if.md), [.WHILE](../intrinsics/.while.md), or [.REPEAT](../intrinsics/.repeat.md) blocks and evaluated at run time, not at assembly time.  
  
## Syntax  
  
```  
  
ZERO?  
  
```
