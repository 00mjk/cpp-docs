# .LISTALL
Starts listing of all statements.  
  
## Syntax  
  
```  
  
.LISTALL  
  
```  
  
## Remarks  
 Equivalent to the combination of [.LIST](../intrinsics/.list.md), [.LISTIF](../intrinsics/.listif.md), and [.LISTMACROALL](../intrinsics/.listmacroall.md).
