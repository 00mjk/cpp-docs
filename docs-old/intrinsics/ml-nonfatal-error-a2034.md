# ML Nonfatal Error A2034
**must be in segment block**  
  
 One of the following was found outside of a segment block:  
  
-   An instruction  
  
-   A label definition  
  
-   A [THIS](../intrinsics/operator-this.md) operator  
  
-   A [$](../intrinsics/$.md) operator  
  
-   A procedure definition  
  
-   An [ALIGN](../intrinsics/align--masm-.md) directive  
  
-   An [ORG](../intrinsics/org.md) directive
