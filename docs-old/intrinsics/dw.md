# DW
Allocates and optionally initializes a word (2 bytes) of storage for each `initializer`. Can also be used as a type specifier anywhere a type is legal. `DW` is a synonym of [WORD](../intrinsics/word.md).  
  
## Syntax  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
