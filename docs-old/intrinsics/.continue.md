# .CONTINUE
Generates code to jump to the top of a [.WHILE](../intrinsics/.while.md) or [.REPEAT](../intrinsics/.repeat.md) block if `condition` is true.  
  
## Syntax  
  
```  
  
.CONTINUE [[.IF condition]]  
```
