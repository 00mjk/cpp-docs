# File Handling Classes
These classes provide methods for handling files, temporary files, and memory-mapped files.  
  
-   [CAtlFile](../atl/catlfile-class.md) This class provides a thin wrapper around the Windows file-handling API.  
  
-   [CAtlFileMapping](../atl/catlfilemapping-class.md) This class represents a memory-mapped file, adding a cast operator to the methods of [CAtlFileMappingBase](../atl/catlfilemappingbase-class.md).  
  
-   [CAtlFileMappingBase](../atl/catlfilemappingbase-class.md) This class represents a memory-mapped file.  
  
-   [CAtlTemporaryFile](../atl/catltemporaryfile-class.md) This class provides methods for the creation and use of a temporary file.
