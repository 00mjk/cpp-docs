# Collection Classes
The following classes provide support for arrays, lists, maps, and also traits methods for helping with comparisons and element access.  
  
-   [CAtlArray](../atl/catlarray-class.md) This class implements an array object.  
  
-   [CAtlList](../atl/catllist-class.md) This class provides methods for creating and managing a list object.  
  
-   [CAtlMap](../atl/catlmap-class.md) This class provides methods for creating and managing a map object.  
  
-   [CAutoPtrArray](../atl/cautoptrarray-class.md) This class provides methods useful when constructing an array of smart pointers.  
  
-   [CAutoPtrElementTraits](../atl/cautoptrelementtraits-class.md) This class provides methods, static functions, and typedefs useful when creating collections of smart pointers.  
  
-   [CAutoPtrList](../atl/cautoptrlist-class.md) This class provides methods useful when constructing a list of smart pointers.  
  
-   [CAutoVectorPtrElementTraits](../atl/cautovectorptrelementtraits-class.md) This class provides methods, static functions, and typedefs useful when creating collections of smart pointers using vector new and delete operators.  
  
-   [CComQIPtrElementTraits](../atl/ccomqiptrelementtraits-class.md) This class provides methods, static functions, and typedefs useful when creating collections of COM interface pointers.  
  
-   [CComSafeArray](../atl/ccomsafearray-class.md) This class is a wrapper for the [SAFEARRAY Data Type](assetId:///9ec8025b-4763-4526-ab45-390c5d8b3b1e) structure.  
  
-   [CComSafeArrayBound](../atl/ccomsafearraybound-class.md) This class is a wrapper for a [SAFEARRAYBOUND](assetId:///303a9bdb-71d6-4f14-8747-84cf84936c6d) structure.  
  
-   [CComUnkArray](../atl/ccomunkarray-class.md) This class stores **IUnknown** pointers and is designed to be used as a parameter to the [IConnectionPointImpl](../atl/iconnectionpointimpl-class.md) template class.  
  
-   [CDefaultCharTraits](../atl/cdefaultchartraits-class.md) This class provides two static functions for converting characters between uppercase and lowercase.  
  
-   [CDefaultCompareTraits](../atl/cdefaultcomparetraits-class.md) This class provides default element comparison functions.  
  
-   [CDefaultElementTraits](../atl/cdefaultelementtraits-class.md) This class provides default methods and functions for a collection class.  
  
-   [CDefaultHashTraits](../atl/cdefaulthashtraits-class.md) This class provides a static function for calculating hash values.  
  
-   [CElementTraits](../atl/celementtraits-class.md) This class is used by collection classes to provide methods and functions for moving, copying, comparison, and hashing operations.  
  
-   [CElementTraitsBase](../atl/celementtraitsbase-class.md) This class provides default copy and move methods for a collection class.  
  
-   [CHeapPtrElementTraits](../atl/cheapptrelementtraits-class.md) This class provides methods, static functions, and typedefs useful when creating collections of heap pointers.  
  
-   [CHeapPtrList](../atl/cheapptrlist-class.md) This class provides methods useful when constructing a list of heap pointers.  
  
-   [CInterfaceArray](../atl/cinterfacearray-class.md) This class provides methods useful when constructing an array of COM interface pointers.  
  
-   [CInterfaceList](../atl/cinterfacelist-class.md) This class provides methods useful when constructing a list of COM interface pointers.  
  
-   [CPrimitiveElementTraits](../atl/cprimitiveelementtraits-class.md) This class provides default methods and functions for a collection class composed of primitive data types.  
  
-   [CRBMap](../atl/crbmap-class.md) This class represents a mapping structure, using a Red-Black binary tree.  
  
-   [CRBMultiMap](../atl/crbmultimap-class.md) This class represents a mapping structure that allows each key to be associated with more than one value, using a Red-Black binary tree.  
  
-   [CRBTree](../atl/crbtree-class.md) This class provides methods for creating and utilizing a Red-Black tree.  
  
-   [CSimpleArray](../atl/csimplearray-class.md) This class provides methods for managing a simple array.  
  
-   [CSimpleArrayEqualHelper](../atl/csimplearrayequalhelper-class.md) This class is a helper for the [CSimpleArray](../atl/csimplearray-class.md) class.  
  
-   [CSimpleArrayEqualHelperFalse](../atl/csimplearrayequalhelperfalse-class.md) This class is a helper for the [CSimpleArray](../atl/csimplearray-class.md) class.  
  
-   [CSimpleMap](../atl/csimplemap-class.md) This class provides support for a simple mapping array.  
  
-   [CSimpleMapEqualHelper](../atl/csimplemapequalhelper-class.md) This class is a helper for the [CSimpleMap](../atl/csimplemap-class.md) class.  
  
-   [CSimpleMapEqualHelperFalse](../atl/csimplemapequalhelperfalse-class.md) This class is a helper for the [CSimpleMap](../atl/csimplemap-class.md) class.  
  
-   [CStringElementTraits](../atl/cstringelementtraits-class.md) This class provides static functions used by collection classes storing `CString` objects.  
  
-   [CStringElementTraitsI](../atl/cstringelementtraitsi-class.md) This class provides static functions related to strings stored in collection class objects. It is similar to [CStringElementTraits](../atl/cstringelementtraits-class.md), but performs case-insensitive comparisons.  
  
-   [CStringRefElementTraits](../atl/cstringrefelementtraits-class.md) This class provides static functions related to strings stored in collection class objects. The string objects are dealt with as references.
