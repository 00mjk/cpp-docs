# &lt;scoped_allocator&gt;
Defines the container template class scoped_allocator.  
  
## Syntax  
  
```cpp  
#include <scoped_allocator>  
```

|||  
|-|-|  
|[operator!=](../stdcpplib/-scoped_allocator--operators.md#operator_neq)|Tests if the scoped_allocator object on the left side of the operator is not equal to the list object on the right side.|  
|[operator==](../stdcpplib/-scoped_allocator--operators.md#operator_eq_eq)|Tests if the scoped_allocator object on the left side of the operator is equal to the list object on the right side.|
