# Linker Tools Error LNK1169
one or more multiply defined symbols found  
  
 The build failed due to multiple definitions of one or more symbols. This error is preceded by error [LNK2005](../othererrors/linker-tools-error-lnk2005.md).  
  
 The [/FORCE](../buildref/-force--force-file-output-.md) or **/FORCE:MULTIPLE** option overrides this error.
