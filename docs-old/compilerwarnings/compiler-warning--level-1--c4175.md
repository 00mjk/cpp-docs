# Compiler Warning (level 1) C4175
\#pragma component(browser, on) : browser info must initially be specified on the command line  
  
 To use [component](../c/component.md) pragma, you must generate browse information during compilation ([/FR](../buildref/-fr---fr--create-.sbr-file-.md)).
