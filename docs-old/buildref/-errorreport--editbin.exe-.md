# /ERRORREPORT (editbin.exe)
```  
/ERRORREPORT[NONE | PROMPT | QUEUE | SEND ]  
```  
  
## Remarks  
 If editbin.exe fails at runtime, you can use **/ERRORREPORT** to send information to Microsoft about these internal errors.  
  
 For more information about **/ERRORREPORT**, see [/errorReport (Report Internal Compiler Errors)](../buildref/-errorreport--report-internal-compiler-errors-.md).
