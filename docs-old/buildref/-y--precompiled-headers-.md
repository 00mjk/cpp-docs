# /Y (Precompiled Headers)
The following compiler options affect the generation and use of precompiled headers:  
  
-   [/Y- (Ignore Precompiled Header Options)](../buildref/-y---ignore-precompiled-header-options-.md)  
  
-   [/Yc (Create Precompiled Header File)](../buildref/-yc--create-precompiled-header-file-.md)  
  
-   [/Yd (Place Debug Information in Object File)](../buildref/-yd--place-debug-information-in-object-file-.md)  
  
-   [/Yl (Inject PCH Reference for Debug Library)](../buildref/-yl--inject-pch-reference-for-debug-library-.md)  
  
-   [/Yu (Use Precompiled Header File)](../buildref/-yu--use-precompiled-header-file-.md)  
  
 For details on working with precompiled headers, see [Creating Precompiled Header Files](../buildref/creating-precompiled-header-files.md).
