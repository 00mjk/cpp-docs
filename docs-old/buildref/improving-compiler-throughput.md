# Improving Compiler Throughput
Use precompiled header files to build your project faster. This is important if you are using ATL, MFC, or the [!INCLUDE[winsdkshort](../atl/includes/winsdkshort_md.md)] header files.  
  
 See [/Yc](../buildref/-yc--create-precompiled-header-file-.md) and [/Yu](../buildref/-yu--use-precompiled-header-file-.md).  
  
 For more information on precompiled headers, see [Creating Precompiled Header Files](../buildref/creating-precompiled-header-files.md).
