# CNoMultipleResults Class
Used as a template argument (*TMultiple*) to [CCommand](../data/ccommand-class.md) to create an optimized command that handles a single result set.  
  
## Syntax  
  
```  
class CNoMultipleResults  
```  
  
## Remarks  
 If you want a command to handle multiple result sets, use [CMultipleResults](../data/cmultipleresults-class.md) instead.  
  
## Requirements  
 **Header:** atldbcli.h
