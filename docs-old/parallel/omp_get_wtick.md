# omp_get_wtick
Returns the number of seconds between processor clock ticks.  
  
## Syntax  
  
```  
double omp_get_wtick( );  
```  
  
## Remarks  
 For more information, see [3.3.2 omp_get_wtick Function](../parallel/3.3.2-omp_get_wtick-function.md).  
  
## Example  
 See [omp_get_wtime](../parallel/omp_get_wtime.md) for an example of using `omp_get_wtick`.
